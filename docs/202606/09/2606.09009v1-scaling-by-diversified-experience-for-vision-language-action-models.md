## 摘要
视觉-语言-动作模型在实际部署中面临重大挑战，原因在于高层推理与低层控制的纠缠以及策略优化的不稳定性。本文提出SyVLA，一种通过多样化经验训练的鲁棒VLA模型。我们提出意图解耦算法，从推理上下文中分离出与控制相关的特征，并采用相似样本引导的强化学习流程来稳定策略更新并缓解分布偏移。在真实机器人任务和多模态基准上的大量实验表明，与现有方法相比，SyVLA实现了更高的任务成功率和更强的分布外泛化能力，同时有效保留了核心视觉-语言能力。代码和数据集在项目页面发布：\href{https://sy-vla.github.io/}{project page}。

## Abstract
Vision-Language-Action models face significant challenges in real-world deployment due to the entanglement of high-level reasoning with low-level control, and the instability of policy optimization. In this paper, we introduce SyVLA, a robust VLA model trained with diversified experiences. We propose an Intention Decoupling algorithm to isolate control-relevant features from reasoning contexts and a similar-sample guided RL pipeline to stabilize policy updates and mitigate distribution shift. Extensive experiments on real-world robotic tasks and multi-modal benchmarks demonstrate that SyVLA achieves superior task success rates and stronger out-of-distribution generalization compared to existing methods, while effectively preserving core vision-language capabilities. Codes and Datasets is released on \href{https://sy-vla.github.io/}{project page}.

## 1. 论文的核心问题与整体含义（研究动机和背景）

*   **问题背景**：视觉-语言-动作模型在真实世界部署时面临两大瓶颈：(1) **推理与控制的纠缠**：VLA 模型过度强调动作学习，导致底层 VLM 的视觉语言理解与推理能力被严重削弱（灾难性遗忘），偏离了人类通过知识迁移实现广泛任务胜任的智能模式。(2) **策略优化不稳定**：基于模仿学习的训练目标（精确拟合每一步专家动作）与真实任务成功（长时域闭环执行）之间存在严重不匹配，导致误差累积和分布偏移；而强化学习虽可缓解，但 VLA 模型的数十亿参数规模和高维连续动作空间使得 RL 极易发生策略漂移甚至能力崩溃。
*   **整体意义**：本文旨在设计一种**平衡任务执行能力与视觉语言理解能力**、且具备**稳定强化学习训练流程**的 VLA 模型。核心思路是通过**多样化经验训练**（同时使用机器人数据和多模态数据）并结合**意图解耦**与**相似样本引导的 RL** 来同时解决上述两大问题。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 2.1 SyVLA 模型架构（双系统）
*   **VLM（高 Level 核心）**：使用 Qwen2.5VL-3B，负责环境感知、语言推理、生成“控制意图”（control intention）。
*   **Action Expert（低 Level 执行）**：一个 0.69B 参数的 Transformer 流匹配模型，以 VLM 输出的控制意图为条件生成连续动作。
*   **连接方式**：在 VLM 输入序列末尾添加一组可学习的 Feature Query Tokens（n=20），其最后一层隐藏状态（Feature Query States）经过 MLP Adapter 后作为控制条件输入 Action Expert。
*   **训练流程**：三阶段：预训练（大规模机器人数据 + 多模态数据，35%混合比）→ 任务微调（数百条任务轨迹 + 多模态数据）→ 强化学习。

### 2.2 意图解耦算法（Intention Decoupling）
*   **问题**：VLM 的高层推理（CoT）信息泄漏到 Feature Query States 中，导致动作专家犹豫不决、精度下降。
*   **方法**：
    1.  第一次前向传播：使用原始 Feature Query States 计算动作损失 \(L_{action}\)。
    2.  计算 \(L_{action}\) 对每个 Feature Query State \(h_i\) 的梯度 \(\mathbf{g}_i = \frac{\partial L_{action}}{\partial h_i}\)。
    3.  统计梯度 \(\ell_2\) 范数分布，将小于第 5 百分位阈值的对应状态置零（mask 掉）。
    4.  第二次前向传播：使用 mask 后的 Feature Query States 重新计算损失并完成更新。
*   **理论分析**（简化单层 Transformer）：梯度范数小的状态对应注意力权重小或值与输出相近（冗余信息），mask 掉可减少控制无关信息，在域内不影响性能，但在分布外泛化时避免非因果决策。
*   **优点**：仅增加约 10% 训练时间（额外计算仅在 Action Expert 上，该模块参数不足 20%）。

### 2.3 相似样本引导的强化学习 Pipeline
*   **动机**：在线 RL 中 VLA 模型极易发生性能崩溃（梯度爆炸、动作不安全）。
*   **方法**：
    *   **相似样本检索**：使用 CLIP 编码器将观测图像映射为特征向量，在 IL 数据集中检索与当前 rollout 样本语义最相似的 top-K 样本。
    *   **混合训练**：将检索到的 IL 样本与 rollout 样本合并为一个 batch 进行 PPO 更新。
    *   **优势值设定**：对检索到的 IL 样本，将其优势值强制设为 1.0（而非使用 GAE 计算），以稳定更新方向并避免价值模型误导；同时将 rollout 样本的 GAE 值 clip 到 [-0.8, 0.8] 以限制置信度。
*   **效果**：可缓解策略漂移，显著提升稀疏奖励长时域任务的成功率。
