## 摘要
基于学习的腿式机器人视觉导航通常依赖于来自层级状态估计的连续目标更新，以提供持久的定向参考。这种依赖带来了额外的感知和计算开销，且偏离了完全的端到端移动自主性。此外，在部分可观测条件下，策略容易学习短视行为，容易陷入死胡同和复杂结构布局中。为解决这些局限，我们研究了一种目标初始化的导航设置，其中目标仅在回合开始时提供一次，要求机器人基于内在空间记忆运行，无需外部模块的后续目标更新。在这项工作中，我们提出了GUIDE，一个完全端到端的强化学习框架，旨在培养内在的方向感知。具体来说，GUIDE包含一个空间锚点预测器，利用多频率本体感受历史提取自运动表示，从而维护用于导航的持久长时域空间上下文。同时，它利用原始深度流感知局部环境几何形状。我们在四足机器人的仿真和真实场景中评估了所提框架。实验表明，GUIDE学习了可靠的自运动与方向感知，使得完全端到端部署的策略能够在没有后续目标指导或先验地图的情况下，安全穿越密集杂乱和结构化迷宫。

## Abstract
Learning-based visual navigation for legged robots typically relies on continuous goal updates from hierarchical state estimation to provide a persistent directional reference. This reliance incurs additional sensory and computational overhead and deviates from fully end-to-end mobile autonomy. Furthermore, under partial observability, policies are prone to learn myopic behaviors, easily becoming trapped in dead ends and complex structural layouts. To address these limitations, we investigate a goal-initialized navigation setting, where the target is provided only once at the beginning of an episode, requiring the robot to operate based on intrinsic spatial memory without subsequent goal updates from external modules. In this work, we propose GUIDE, a fully end-to-end reinforcement learning framework designed to cultivate internal directional awareness. Specifically, GUIDE incorporates a spatial anchor predictor that leverages multi-frequency proprioceptive history to extract egomotion representations, thereby maintaining a persistent long-horizon spatial context for navigation. Concurrently, it utilizes raw depth streams to perceive local environmental geometry. We evaluate the proposed framework across both simulation and real-world scenarios on a quadruped robot. Experiments show that GUIDE learns reliable egomotion and directional awareness, enabling a fully end-to-end deployed policy to safely navigate through dense clutter and structured mazes without subsequent goal guidance or prior maps.

- **意义**：提出“目标初始化导航”新设定，推动腿式机器人向更纯粹、更自主的端到端导航迈进，降低对定位、建图等上游模块的依赖。

## 2. 方法论

### 核心思想
- 提出 **GUIDE**（Goal-Initialized Directional Understanding for End-to-End Visual Navigation），一种完全端到端的强化学习框架。
- 在训练中引入**空间锚点预测器**，利用多频率本体感受历史监督预测身体速度、相对目标位置、相对出生位置，迫使策略习得持续的空间上下文和方向感知。
- 结合原始深度流感知局部几何，最终仅通过单次初始目标、当前观测和历史信息输出控制指令。

### 关键技术细节
- **输入**：
  - **本体感受**：分为低频（10Hz，N=10步）和高频（200Hz，M=20步）两条时间流。低频捕获宽时间跨度的历史运动上下文，高频捕获细粒度瞬态动力学。
  - **深度图像**：8帧历史缓冲区，每帧经噪声注入、裁剪、下采样至30×43。
  - **初始目标**：仅在回合开始时提供一次（egocentric 3维向量）。
- **网络结构**（图2）：
  1. 高频流经帧编码器和自注意力生成近期运动嵌入。
  2. 与低频流编码拼接，再经自注意力+GRU得到本体感受token `z_p`。
  3. **空间锚点预测**：三个轻量MLP头从`z_p`分别预测身体速度、相对目标位置、相对出生位置，监督损失为Huber loss。
  4. 深度帧经共享CNN编码+时间自注意力得到视觉token `z_d`。
  5. 用交叉注意力以`z_p`为query、`z_d`为key/value融合，再经GRU得到空间隐变量`z_s`。
  6. Actor以`[z_s, 预测的速度、目标、出生, 当前本体感受]`为输入，输出twist命令（vx, vy, ωz）。
- **奖励设计**：
  - 稀疏任务奖励：位置到达、航向到达，触发后持续一段时间才算成功；失败终止惩罚。
  - 密集进展奖励：欧氏距离进展、迷宫单元距离进展；近终点速度阻尼；静止惩罚；碰撞惩罚等。
- **多批评家（MuC）**：将总奖励分为稀疏任务组和密集成形组，分别用独立价值网络估计，并统一标准化优势估计，提升训练稳定性。
- **训练**：使用PPO，异步演员-批评家，批评家拥有特权信息（占据图、探索图两张60×60地图）。
- **课程学习**：地形难度逐步增加（杂乱地形增加障碍密度，迷宫地形扩大网格尺寸）；目标距离逐步增大。

## 3. 实验设计

### 仿真 Benchmark
- **场景**：两类程序化生成的地形：
  - **杂乱环境**：随机布置窄巷、L型拐角、柱子等，造成频繁遮挡和局部陷阱。
  - **迷宫环境**：基于深度优先搜索生成的完美迷宫，无环，每个起-终点对只有唯一路径。
  - 各分 Easy / Hard 两个难度等级，共四个设置。
- **评价指标**：成功率(SR)、到达时间(AT)、碰撞率(CR)。每项评价 1000 个 episodes，跨越 50 个随机布局。
- **对比方法**：
  - GUIDE（全模型）
  - 消融变体：去除高频本体感受、去除目标预测头、去除出生预测头、去除速度预测头、去除多批评家。
  - 两个 Oracle 变体：GUIDE-GT-Inference（推理时用真值替换预测的目标和出生）、GUIDE-GT-Train（训练和推理均使用连续真值目标与出生）。

### 真实世界实验
- **室内实验室**（8m×8m）：杂乱障碍环境和迷宫环境。机器人根据内部估计距离自动停止。额外指标：最终距离（FD，实际物理距离）。
- **户外野外**：长办公室走廊、草地、密集植被、动态障碍物等，定性验证零样本迁移。
- **定量**：比较 GUIDE 与两个最高成功率的消融变体，每种10次独立试验。

## 6. 主要结论与发现

- GUIDE 成功实现**仅初始目标一次**的端到端视觉导航，可靠地在密集杂乱环境和结构迷宫内安全导航并逃离死胡同。
- **高频本体感受**至关重要：缺失后策略几乎完全失效（成功率接近0%），且无法保持内部目标的跟踪。
- 三个**空间锚点预测头**共同塑造空间上下文：去除速度预测头导致最大性能下降，表明瞬时运动线索和长时程位置锚点缺一不可。
- **多批评家架构**显著优于单批评家，加速收敛并达到更高奖励。
- GUIDE 与 Oracle 变体性能差距很小（成功率近似，到达时间略长），证实策略已有效学习并利用内部空间上下文。
- 真实世界零样本迁移成功，在室内和野外均表现出鲁棒导航能力。

## 7. 优点

- **完全端到端**：无需地图、无需连续目标更新、无需额外定位模块，真正降低系统复杂度。
- **新颖的目标初始化设定**：推动研究更接近生物导航中的认知地图概念。
- **空间锚点预测**：利用本体感受历史监督预测关键导航变量，使策略习得内在空间记忆，思路简洁有效。
- **多频率本体感受处理**：捕获长期运动趋势与短期瞬态动力学，是性能关键。
- **多批评家**：有效处理稀疏+密集混合奖励，提升训练稳定性。
- **课程学习与丰富的程序化地形**：促使策略同时具备反应式避障和长距离推理能力。
- **充分的仿真消融**：系统验证各组件贡献，并对比 Oracle 上限，实验设计严谨。
