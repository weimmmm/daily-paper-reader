## 摘要
动态环境中的终身具身导航要求机器人从零散的观测中形成持久的场景理解，这对于依赖显式地图或场景图且难以泛化到结构化环境之外的现有方法来说仍然困难。我们提出AllDayNav，一种终身自学习导航框架，通过强化学习将场景动态隐式编码到大模型数十亿规模的参数中，并由自进化的多模态记忆驱动，该记忆维护并更新视觉关键帧、语义描述和时间上下文，同时自主生成开放词汇指令、图像目标和结构化奖励。在跨房间、跨回合和跨任务场景下的合成及真实环境实验表明，AllDayNav实现了接近100%的成功率，并在路径效率和鲁棒性方面持续超越基于地图、VLM和强化学习的强大基线，展示了隐式、记忆驱动的强化学习作为可靠终身导航的可扩展替代方案，以替代显式地图构建。

## Abstract
Lifelong embodied navigation in dynamic environments requires robots to form persistent scene understanding from fragmentary observations, which remains difficult for existing methods that rely on explicit maps or scene graphs and struggle to generalize beyond structured settings. We propose AllDayNav, a lifelong self-learning navigation framework that implicitly encodes scene dynamics into the billion-scale parameters of a large model via reinforcement learning, powered by a self-evolving multimodal memory that maintains and updates visual keyframes, semantic descriptions, and temporal context while autonomously generating open-vocabulary instructions, image goals, and structured rewards. Experiments in both synthetic and real-world environments across cross-room, cross-episode, and cross-task scenarios show that AllDayNav achieves success rates approaching $100\%$ and consistently surpasses strong map-based, VLM, and RL baselines in path efficiency and robustness, demonstrating implicit, memory-driven reinforcement learning as a scalable alternative to explicit mapping for reliable lifelong navigation.

## 1. 核心问题与整体含义（研究动机和背景）
**终身具身导航**要求机器人在动态、未知环境中长时间运行，需要从片段式观测中构建持久的场景理解，并适应环境变化。现有方法存在两个关键缺陷：
- **记忆不足**：VLA导航系统仅依赖短时帧级记忆、静态场景图或有限交互，缺乏跨回合的持续记忆；
- **适应能力差**：策略更新局限于单回合或少量交互，无法累积知识和长期优化。

受认知科学中视觉心理意象的启发，论文提出 **AllDayNav**，旨在通过**隐式记忆 + 强化学习**将场景动态编码到大模型参数中，实现无需人类监督的终身自学习导航，作为显式建图的可扩展替代方案。

## 2. 论文提出的方法论

### 核心思想：记忆-策略协同进化
- 构建一个**自进化的多模态记忆数据库**，在线存储视觉关键帧、语义描述、时间戳和嵌入向量。
- 利用记忆**自主生成训练任务**（指令-目标图像对），驱动策略通过**保守Q学习（CQL）** 持续优化。
- 形成一个正反馈循环：探索 → 丰富记忆 → 生成更优任务 → 提升策略 → 更高效探索。

### 关键技术细节

#### 2.1 预训练的VLA导航骨干
- **视觉编码**：双编码器（SigLIP 提取语义 + DINOv2 提取结构），特征拼接后经MLP投影。
- **观测序列**：当前帧和图像目标全分辨率（各64 token），历史帧经全局平均池化压缩，并施加指数衰减注意偏置。
- **动作表示**：预测8个连续航点 `(Δx, Δy, Δθ)`，经4层MLP动作头输出。
- **预训练**：在Habitat中收集5万条轨迹（80%专家轨迹+20%探索轨迹），使用MSE损失监督航点预测，并应用强数据增强（模糊、抖动、裁切、噪声等）缩小sim-to-real差距。

#### 2.2 自进化多模态记忆数据库
- **条目结构**：`(RGB图像, 语义描述, 时间戳, LLaVE嵌入)`。
- **自主构建**：每5步评估候选帧，与上一关键帧嵌入余弦相似度<0.15时触发记忆更新。
- **去重与替换**：计算新帧与现有记忆的最大相似度；若≤0.7判定为新场景，调用VLM(Qwen3-VL-8B)生成描述并插入；若>0.7则检查图像-文本对齐度，保留更优的视觉实例。
- **VLM语义检索**：给定自然语言指令，VLM从记忆描述中选择最佳匹配条目并输出置信度，支持时间敏感指令。

#### 2.3 自指令生成与混合目标检索
- **指令生成**：每10回合或记忆增长20%时，VLM基于记忆描述生成50条多样指令（物体级、功能级、空间级、组合级、时间级）。
- **混合检索**：优先从记忆检索（置信度≥0.6），否则回退到外部互联网图像集合；在记忆稀疏阶段以概率`p_fallback = max(0, 1 - |M|/20)`触发回退，确保训练信号持续。

#### 2.4 保守Q学习（CQL）在线优化
- **状态**：`(当前帧, 目标图像, 历史帧压缩)`；**动作**：8航点；**折扣因子**γ=0.99。
- **奖励**：终端成功奖励+10（相似度≥0.75）、进度奖励（首次跨越0.5/0.6/0.7阈值各+2）、时间惩罚-0.01、碰撞惩罚-0.1。
- **CQL损失**：在贝尔曼误差基础上增加保守正则化项，抑制离群动作的Q值高估，α=5.0。
- **策略损失**：Q值最大化 + 行为克隆正则化（λ_bc=0.1），使用成功轨迹的子集。

#### 算法流程（Algorithm 1）
1. 并行运行探索循环和策略优化循环；
2. 探索循环：自指令生成→混合检索→在线探索→记忆更新；
3. 优化循环：每100步从回放缓冲区采样，更新Critic（LCQL）和Policy（Lpolicy）。

## 6. 主要结论与发现
1. **AllDayNav在仿真和真实环境中成功率接近100%**，在SPL和回合长度上也显著优于所有基线。
2. **记忆持久性、自指令生成、保守在线更新三者缺一不可**：移除任一组件（无跨回合记忆、无自指令、无BC损失）均导致性能明显下降。
3. **适应动态环境**：场景突变后性能快速恢复并重新收敛，而“无跨回合记忆”变体无法恢复。
4. **检索准确率高**：人工评估显示检索感知成功率与标准成功率差距很小（约1-2%），证明VLM语义检索可靠。
5. **sim-to-real有效**：通过强数据增强和航点预测，策略成功迁移到真实机器人，无需额外模型适配。

## 7. 优点
- **完全自主**：无需人类标注、演示或干预，实现真正的终身自学习。
- **创新性记忆设计**：多模态自进化记忆结合VLM语义检索，支持时间敏感和复杂指令。
- **混合目标检索**：记忆优先+互联网回退，解决了冷启动问题并驱动探索。
- **稳定的在线RL**：CQL+BC正则化防止过估计和遗忘，适合混合质量数据。
- **全面实验验证**：覆盖仿真/真实、标准/挑战场景、消融分析，实验设计严谨。
- **可扩展性**：隐式记忆直接编码到大模型参数，避免显式建图的更新瓶颈。

## 8. 不足与局限
