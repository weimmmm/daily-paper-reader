## 摘要
事后重标记通常将已到达的未来状态转化为精确的目标，当任务成功仅依赖于状态的子集时，这会过度约束离线机器人学习。我们提出目标集事后重标记（GS-HER），这是HER的谓词级泛化，其中已到达的状态验证查询定义的目标集，而非单一目标状态。一个二元查询指定哪些变量定义成功，使目标谓词成为推理时的输入，同时保持底层离线GCRL算法不变。在OGBench任务和五个离线目标条件学习器上，当全状态目标受干扰维度瓶颈影响时，GS-HER提升了性能，并将事后重标记转化为可复用的目标接口：一个检查点可以回答多个机器人目标谓词而无需重新训练。

## Abstract
Hindsight relabeling usually turns achieved future states into exact goals, which can overconstrain offline robot learning when task success depends only on a subset of the state. We propose Goal-Set Hindsight Relabeling (GS-HER), a predicate-level generalization of HER in which achieved states certify query-defined goal sets rather than singleton goal states. A binary query specifies which variables define success, making the goal predicate an inference-time input while leaving the underlying offline GCRL algorithm unchanged. Across OGBench tasks and five offline goal-conditioned learners, GS-HER improves performance when full-state goals are bottlenecked by nuisance dimensions and turns hindsight relabeling into a reusable goal interface: one checkpoint can answer multiple robot goal predicates without retraining.

## 1. 核心问题与整体含义
- **研究动机**：离线机器人学习（Offline GCRL）中，标准事后重标记（HER）将轨迹中未来状态作为精确目标（即全状态匹配），导致任务成功仅依赖部分状态变量（如物体位置）时，无关维度（如机器人构型、速度、夹爪状态）成为人工约束，过度限制学习。
- **整体含义**：提出目标集事后重标记（GS-HER），将目标从单一状态泛化为由二元查询（Query）定义的目标集，使目标谓词成为推理时输入，从而在不改变底层离线GCRL算法前提下，使同一检查点可应答多种目标谓词，极大提升目标灵活性。

## 2. 方法论
- **核心思想**：已到达的状态不仅可认证自身，还可认证任何包含它的目标集。通过二元查询 q ∈ {0,1}^D 指定哪些状态变量定义成功（活动坐标），其余为干扰变量，从而定义目标集 G(g,q) = {s' : q⊙s' = q⊙g}。
- **关键技术细节**：
  - 引入查询条件化输入 ψ(s,g,q) = [q⊙s + (1-q)⊙e_s, q⊙g + (1-q)⊙e_g, q]，其中 e_s, e_g 为干扰坐标嵌入（可学习）；查询 q 显式说明哪些坐标定义成功。
  - 成功谓词 c_{g,q}(s') = I[ d_q(s',g) ≤ ε_q ]，其中 d_q 为归一化平方距离。
  - GS-HER 作为包装器（wrapper）应用于现有离线 GCRL 算法（如 GCBC、CRL、GCIVL、GCIQL、GCDL），仅改变重标号的目标表示和成功标记，学习目标不变。
- **算法流程**（Algorithm 1）：从数据集中采样轨迹和过渡，采样参考目标 g 和查询 q，构造 ψ 输入，使用查询条件化成功标记更新底层学习器。推理时通过改变 q 即可切换目标谓词。

## 6. 主要结论与发现
- GS-HER 在操纵任务上显著优于 HER-Full（平均成功率从 24.5% 提升至 55.0%，接近 oracle 的 60.6%），尤其在干扰维度占主导时（相关性 r=0.87）。
- 单一 GS-HER 检查点可通过改变查询回答多种目标谓词（如物体位置、物体偏航、末端执行器位姿等），无需重新训练，而 HER-Task 每种谓词需重训。
- GS-HER 学到的距离估计可跟踪任务相关进度，而 HER-Full 因干扰维度饱和。
- 在导航任务（PointMaze）中，全状态与任务状态对齐，GS-HER 无优势，符合预期。

## 7. 优点
- 方法简洁：只需改变重标号语义和输入表示，不修改底层学习器，便于集成。
- 实现通用：对多种离线 GCRL 算法（模仿、对比、值函数）有效，证明了正交性。
- 灵活性：单模型支持推理时查询不同目标谓词，避免为每个任务重训。
- 诊断清晰：通过距离学习器可视化直观展示了干扰维度瓶颈。
- 实验设计公平：与 oracle 投影基线对比，揭示了干扰维度影响。
