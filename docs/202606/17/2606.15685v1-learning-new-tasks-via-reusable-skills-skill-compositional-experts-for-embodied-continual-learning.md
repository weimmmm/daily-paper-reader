## 摘要
具身持续学习（ECL）旨在使机器人能够在闭环控制下持续获取新的操作任务，同时保留先前学习的行为。与传统的持续学习相比，ECL遭受更严重的灾难性遗忘。闭环控制下积累的特征漂移通过顺序决策逐步传播，导致先前学习的行为退化。ECL的一个关键挑战在于如何在不断演变的任务中进行结构化技能重用，因为现有方法主要关注技能学习，而没有明确组织技能以实现连贯的任务执行。为解决这一问题，我们提出了SCE，一种用于ECL的技能组合专家框架。SCE通过组合技能基础（CSG）构建技能库，将任务示范分解为可重用技能。基于此，双执行与转换专家（DETE）通过技能组合实现新任务学习，其中一个分支确保技能执行，另一个分支支持技能之间的转换以实现连贯行为。在LIBERO基准测试和真实世界操作任务上的实验表明，SCE持续提高了任务保持率和整体性能。进一步的特征漂移分析和消融研究验证了我们方法的有效性。项目网站：https://eqcy.github.io/sce/。

## Abstract
Embodied Continual Learning (ECL) aims to enable robots to continually acquire new manipulation tasks while retaining previously learned behaviors under closed-loop control. Compared with conventional continual learning, ECL suffers from more severe catastrophic forgetting. Feature drift accumulated under closed-loop control progressively propagates through sequential decision-making, leading to degradation of previously learned behaviors. A key challenge in ECL lies in structured skill reuse across continually evolving tasks, since existing methods primarily focus on skill learning without explicitly organizing them for coherent task execution. To address this issue, we propose SCE, a Skill-Compositional Experts framework for ECL. SCE builds a skill base via Compositional Skill Grounding (CSG), which decomposes task demonstrations into reusable skills. Based on this, Dual Execution-and-Transition Experts (DETE) enable new task learning through skill composition, where one branch ensures skill execution and the other supports transitions between skills for coherent behavior. Experiments on LIBERO benchmarks and real-world manipulation tasks demonstrate that SCE consistently improves retention and overall task performance. Further feature drift analyses and ablation studies verify the effectiveness of our method. Project website: https://eqcy.github.io/sce/.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题定义**：具身持续学习（Embodied Continual Learning, ECL）要求机器人能够在与环境的闭环交互中持续学习新任务，同时不遗忘旧任务。相比传统持续学习，ECL面临更严重的灾难性遗忘，因为闭环控制下的特征漂移会通过时序决策累积传播，最终破坏先前学到的任务。
- **现有局限**：已有ECL方法主要在任务级或技能级缓解遗忘，但缺乏对可重用技能的显式组织。任务级方法（如MoILE）将每个任务视为孤立单元，限制了跨任务模式共享；技能级方法（如IsCiL、LOTUS）虽引入技能原语，但未结构化建模技能之间的组合关系，导致新任务执行时技能重用不一致，放大特征漂移。
- **核心动机**：提出一种“组合式技能重用”视角——每个新任务可表示为已有可重用技能的序列。通过显式构建技能库并设计技能组合机制，既能稳定旧任务执行，又能灵活适应新任务，从而缓解灾难性遗忘。

## 2. 方法论

### 核心思想：Skill-Compositional Experts (SCE) 框架

SCE基于预训练的VLA模型（π0），在动作解码器上附加DETE模块，通过两阶段实现技能组合式ECL：

#### 2.1 组合技能基础（Compositional Skill Grounding, CSG）

- **阶段1——状态感知任务分解**：利用机器人状态动态规则进行时序分割：
  - 规则1：夹爪状态变化处标记边界；
  - 规则2：末端执行器运动趋势突变处标记边界。
  将完整演示分割为若干时序片段。
- **阶段2——VLM引导技能归纳**：对每个片段，使用视觉语言模型（VLM，Qwen3-VL）结合当前技能库B，生成候选技能标签；若技能库中无匹配则新增为可重用技能，否则选择已有技能。最终为每个样本分配技能标注，构建技能库B（K维）。

#### 2.2 双执行与转换专家（Dual Execution-and-Transition Experts, DETE）

- **技能空间感知**：将条件token c（视觉-语言+机器人状态）映射为K维技能分布p = softmax(MLP(c))，并用交叉熵损失监督（训练时利用技能标注）。
- **执行专家分支（Execution Expert Branch）**：维护K个专家{E_k}，每个对应一个技能。通过技能分布p的argmax选择当前最相关技能专家，输出Δy_exe。更新仅作用于被选专家，减少对无关技能干扰。
- **转换专家分支（Transition Expert Branch）**：维护N个专家{G_j}，通过转换感知路由（注入p）计算混合权重u，输出Δy_tr = Σ u_j G_j(y)。建模跨技能转换处的依赖。
- **自适应融合模块**：利用转换感知特征h计算平衡系数α = σ(W_α h)，融合两分支输出Δy = αΔy_exe + (1-α)Δy_tr。在技能边界处α降低以增强转换贡献。

#### 2.3 优化目标

整体损失：L = L_flow + λ L_skill，其中L_flow为预训练VLA策略的流匹配损失，L_skill为技能分布交叉熵，λ为平衡超参。

## 6. 主要结论与发现

- SCE在LIBERO-Goal和LIBERO-Long上取得最佳NBT（4.3%和0.5%）、AUC（82.5和70.0）和Final SR（83.4%和73.4%），大幅优于Task-MoE等基线，遗忘显著降低。
- 真实世界中SCE的Final SR达83.3%，NBT为0.0%，远超其他方法。
- 特征漂移分析显示SCE在动作层特征上漂移远小于Sequential（约5.7-6.9倍差异），闭环状态下状态偏差也更小。
- 消融实验表明CSG和两个专家分支均不可或缺，移除任一成分均导致性能下降；α自适应融合机制在技能转换处有效增强转换分支贡献。

## 7. 优点

- **方法创新**：首次将任务执行显式建模为可重用技能的组合，既有执行专家稳定技能内行为，又有转换专家处理跨技能过渡，设计合理。
- **结构清晰**：CSG将VLM用于技能归纳自动化，无需人工标注；DETE轻量适配器结构保持预训练模型主干不变，便于扩展。
- **实验全面**：涵盖模拟和真实场景、多种基线、多指标、特征漂移量化分析，验证了闭环下优势。
- **性能突出**：在LIBERO基准上遗忘率降低至4.3%（Goal）和0.5%（Long），最终成功率提升至83.4%和73.4%，显著优于现有ECL方法。
