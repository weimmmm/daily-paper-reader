## 摘要
视觉导航基础模型（VNMs）承诺提供端到端学习的导航策略，能够跨不同载体和环境进行零样本部署。为了保持通用性，许多基于视觉的导航模型预测归一化的动作。然而，这种归一化引入了一个关键的部署脆弱性：对相同的归一化轨迹应用不同的缩放因子会改变其物理几何形状，从而降低导航性能并增加碰撞风险。我们通过将模型在归一化动作历史与图像观测上进行条件化来解决这一脆弱性，为模型预测与机器人实际物理位移之间的关系提供明确的上下文。此外，当前的VNMs通常在缺乏明显特征的视觉重复环境中表现不佳。为了解决这个问题，我们集成了DINOv3编码器，其更丰富的表示使我们的模型能够捕捉观测之间的空间和几何维度。VISTA对外部环境具有强大的泛化能力，在户外、森林和办公室场景的零样本实际部署中实现了100%的目标预测准确率，平均跨越95%的检查点，在未见环境中展示了稳定路径跟踪能力。

## Abstract
Vision Navigation Foundation Models (VNMs) promise end-to-end learned navigation policies capable of zero-shot deployment across diverse embodiments and environments. To maintain generality, many vision-based navigation models predict normalized actions. However, this normalization introduces a critical deployment vulnerability: applying different scaling factors to the same normalized trajectory alters its physical geometry, which degrades navigation performance and increases collision risks. We address this vulnerability by conditioning the model on normalized action histories alongside image observations, providing explicit context on the relationship between the model's predictions and the robot's actual physical displacement. Furthermore, current VNMs often struggle in visually repetitive environments that lack distinct features. To resolve this issue, we integrate a DINOv3 encoder, whose richer representations enable our model to capture both spatial and geometric dimensions between observations. VISTA generalizes robustly to out-of-distribution environments, achieving 100% goal prediction accuracy in zero-shot, real-world deployment in Outdoor, Forest and Office settings, and an average of 95% checkpoints crossed, demonstrating consistent path following in unseen environments.

## 1. 核心问题与整体含义（研究动机与背景）

- **核心问题**：现有视觉导航基础模型（VNMs）为了保持对不同机器人载体（embodiment）和环境的通用性，通常输出**归一化动作**（normalized actions）。但在部署时，由于机器人速度、控制频率等差异，施加的**缩放因子（scaling factor）** 不同，会导致相同的归一化轨迹在物理空间中几何形状扭曲，从而**降低导航性能、增加碰撞风险**。模型仅依赖图像序列难以推断隐含的物理步长，造成尺度模糊（metric misalignment）。
- **背景**：VNMs（如ViNT、NoMaD、MetricNet）通过大规模数据集学习通用表示，能够零样本部署。但归一化带来的尺度不一致是阻碍实际部署的关键缺陷。此外，在视觉重复的环境（如走廊、雪地）中，模型因缺乏强特征而表现不佳。
- **整体目标**：提出**VISTA（Scale-Aware Visual Navigation via Action History Conditioning）**，通过**归一化动作历史条件**提供显式的尺度上下文，同时采用**DINOv3**编码器增强视觉几何与语义表示，实现零样本下对速度和控制频率鲁棒的尺度感知导航。

## 2. 方法论：核心思想、关键技术细节

### 2.1 核心思想
- **归一化动作历史条件**：将过去连续归一化动作（机器人位移÷部署步长）作为模型的额外输入，让模型能够从视觉观测与动作历史的对应关系中推断当前的实际物理步长，从而输出与尺度对齐的度量轨迹，而无需牺牲跨载体通用性。
- **DINOv3视觉编码**：使用DINOv3 ViT-S自监督模型提取更强几何与语义特征，改善在特征贫乏环境下的表示能力。

### 2.2 关键技术细节
- **模型架构**（见图6）：
  - 输入：历史观测图像序列 \( O_t \)、目标图像 \( o_g \)、归一化动作历史 \( H_t = \{a_{t-k+1}, \dots, a_{t-1}\} \)。
  - DINOv3将图像转为patch tokens，经投影降采样（2×）。
  - 动作历史经MLP编码为相同维度的token。
  - tokens与用于预测距离和动作的readout tokens拼接，送入Transformer（6层、8头、512维）。
  - 输出：归一化路径点序列 \( \hat{w}_{1:T} \) 和归一化度量距离 \( \hat{d} \)。
- **训练目标**：
  - 多任务损失：\( \mathcal{L}_{\text{total}} = \lambda \mathcal{L}_{\text{action}} + \mu(1-\lambda)\mathcal{L}_{\text{distance}} \)，其中 \(\lambda=0.5, \mu=0.01\)。
  - \(\mathcal{L}_{\text{action}}\)：MSE，预测归一化路径点与真实归一化路径点（\( w_{ij}/\phi_i - \hat{a}_{ij} \)）。
  - \(\mathcal{L}_{\text{distance}}\)：MSE，预测归一化目标距离与真实归一化距离（\( \hat{d}_i - \|p_t - p_g\|/\phi_i \)）。
- **训练数据增强**：为提升对不同步长的鲁棒性，训练时对数据集进行变频率采样（2–12Hz），而传统模型仅用固频（3–4Hz）。
- **动作历史定义**：\( a_i = \Delta p_i / \phi_{\text{deploy}} \)，即归一化的过去位移。模型借此学习视觉位移与步长的关系。

## 6. 主要结论与发现

1. **尺度一致性**：VISTA在离线测试中交叉轨迹误差显著低于ViNT和NoMaD，动作历史进一步提升了预测精度（表1）。
2. **零样本泛化**：VISTA在Outdoor、Forest、Office-Lab环境中均实现**100%成功目标预测**（5/5或3/3），路径长度接近参考路径，平均95%以上检查点通过。而ViNT、NoMaD常出现偏离、假阳性目标或碰撞。
3. **动作历史关键性**：VISTA w/o AH虽在路径跟随上较好，但在需要精确速度调节的场景（如Office-Lab窄门）中全部碰撞，说明单独增加强编码器不足以解决尺度问题。
4. **变步长鲁棒性**：VISTA在0.4m/s/4Hz至0.1m/s/4Hz的大范围内成功完成锐弯，ViNT在低速度下全部失败（表3）。
5. **扰动鲁棒性**：动态门关闭时VISTA能停止等待，后续恢复；环路实验5圈均成功，体现距离预测的一致性。
6. **深度图像零样本迁移**：VISTA在深度输入下损失最低，表明DINOv3学到了跨模态几何特征。

## 7. 优点

- **创新性地使用归一化动作历史**解决了尺度模糊问题，无需直接输入比例标量，处于同一表示空间避免跨空间映射难题。
- **DINOv3编码器**提供强语义几何表示，在视觉重复场景中胜过EfficientNet等传统编码器。
- 训练时**变频率采样**增强了模型对不同速度和控制频率的适应能力。
- **丰富的真实世界评估**覆盖多种典型挑战（开阔地、密林、办公走廊、窄门、动态障碍），且包含消融和扰动测试，实验结果扎实。
- 代码将开源，促进可重复性。

## 8. 不足与局限

1. **缺乏机器人物理尺寸知识**：模型不知道自身宽度、高度，无法显式判断可穿越性（如楼梯、窄间隙）。未来可结合可穿越地图。
2. **视觉相似样本下的困惑**：当连续历史图像几乎一致时，模型难以仅从视觉推断运动大小。建议未来引入自适应拓扑图采样。
3. **极低控制频率（≤1Hz）失败**：在步长极大（0.4m）时VISTA性能下降（成功0/3），说明极端尺度下仍需显式几何先验。
4. **未报告训练资源**：缺少GPU型号、数量、时间等信息，妨碍其他研究者复现。
5. **MetricNet实现潜在偏差**：作者自行实现MetricNet（因无公开代码），可能未完全复现原版性能，对比公平性略有降低。
6. **环境类型覆盖有限**：虽多，但未涉及雪地、沙漠、水下等极端环境；导航距离最长仅36米，大规模环境性能未知。
