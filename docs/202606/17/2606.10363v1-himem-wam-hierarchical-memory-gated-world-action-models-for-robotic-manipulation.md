## 摘要
世界动作模型（WAMs）已成为具身智能的一种强大新范式，通过学习与动作相关的视觉动态，显著增强了泛化能力和鲁棒性。然而，现有的WAMs在长时程机器人操作中仍面临任务相关记忆的挑战。为此，我们提出HiMem-WAM，一种分层记忆门控世界动作模型，融合了以运动为中心的潜在动作、高层技能隐变量以及边界触发的记忆更新。具体而言，我们开发了一个分层潜在动作框架，联合学习底层运动和高层技能隐变量，提供了结构化的时间抽象。同时，一个边界感知的记忆门在预测的技能转换处写入紧凑的任务状态，使得无需在测试时生成未来视频或光流估计即可进行因果推理。在LIBERO、LIBERO-PLUS、RMBench以及真实世界任务上的评估表明，HiMem-WAM的分层隐变量提高了部署扰动下的鲁棒性，而记忆模块显著有利于依赖记忆的长时程操作。

## Abstract
World Action Models (WAMs) have emerged as a new powerful paradigm for embodied intelligence, learning action-relevant visual dynamics that significantly enhance generalization and robustness. However, existing WAMs still struggle with task-relevant memory in long-horizon robotic manipulation. To address this, we present HiMem-WAM, a Hierarchical Memory-Gated WAM that integrates motion-centric latent actions, high-level skill latents, and boundary-triggered memory updates. Specifically, we develop a hierarchical latent action framework that jointly learns low-level motion and high-level skill latents, providing structured temporal abstraction. Meanwhile, a boundary-aware memory gate writes compact task states at predicted skill transitions, enabling causal inference without test-time generation of future video or optical flow estimation. Evaluated on LIBERO, LIBERO-PLUS, RMBench and real-world tasks, HiMem-WAM shows that hierarchical latents improve robustness under deployment perturbations, and the memory module substantially benefits memory-dependent long-horizon manipulation.

## 2. 方法论
### 核心思想
- 将策略分解为三个角色：**高层技能选择**（planning）、**高层技能展开为短时运动**（execution）、**运动解码为具体控制**（decoding）。
- 引入**分层潜在动作框架**：联合学习低层运动潜在变量（low-level latent actions）和高层技能潜在变量（high-level skill latents），提供结构化的时间抽象。
- 设计**边界感知记忆门**：仅在预测的技能转换处写入紧凑任务状态，读取历史上下文以条件化规划器，实现稀疏记忆更新和因果推理。

### 关键技术细节
- **低层潜在动作**：使用多视图光流（DPFlow）计算运动特征，通过变分分词器编码短时上下文，重建光流并与实际动作弱对齐。
- **高层技能潜在**：通过对低层序列进行动态分块（Chunking），学习边界预测器，将可变长度段池化为下一级令牌，最终得到技能序列。
- **记忆门控模块**：
  - 读取门：从外部记忆库检索相关上下文，通过门控注入状态。
  - 写入门：基于预测边界分数决定是否存储当前状态（包括观察、技能、池化的低层潜在块）。
  - 更新操作：仅当写入门超过阈值 η 时追加记忆令牌，并压缩到固定预算。
- **训练流程**：三阶段。
  - Stage I：低层潜在动作分词器训练（光流重建+动作对齐+KL散度）。
  - Stage II：冻结分词器，发现技能边界，预训练规划器和执行器（无记忆）。
  - Stage III：激活记忆门控，微调全策略，使用动作标签监督，写入门由边界标签监督。
- **推理**：因果，仅需当前RGB、本体感受、指令和记忆库，无需生成未来视频或光流。

## 6. 主要结论与发现
- **关键发现1**：分层潜在动作提升了部署扰动下的鲁棒性。在LIBERO上+1.1%，在LIBERO-PLUS零样本设置下+3.8%，表明潜在动作捕获了可迁移的运动模式。
- **关键发现2**：门控记忆有效改善任务状态追踪。在RMBench上总平均成功率达26.3%，M(1)任务达31.6%，显著优于基线（π0.5 10.8%, X-VLA 9.8%）。但在M(n)任务上仅19.8%，弱于专门设计的Mem-0，表明多次记忆推理仍需加强。
- **关键发现3**：真实世界中，HiMem-WAM在Medium任务上比π0.5高12.5%（ST），Hard任务上高25.0%（ST），分层预训练和记忆模块共同改善了长时程操作。
- **关键发现4**：Stage II分层潜在预训练在不同动作表示（关节位置、末端执行器位姿）下均有效，且更难的任务获益更大。
