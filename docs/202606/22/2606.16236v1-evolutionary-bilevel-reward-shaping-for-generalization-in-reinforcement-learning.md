## 摘要
强化学习（RL）在部署到与训练环境不同的环境中时，往往会遇到性能下降的问题。现有技术如领域随机化（DR）可以缓解这一问题，但需要访问多样化的训练环境和完整的轨迹观测性，这些假设在隐私保护或受限场景中无法满足，因为在这些场景中只能获得标量性能指标。我们提出了一种通过进化奖励塑造实现泛化（GERS）的双层优化方法，仅利用来自验证环境的标量反馈来改善在未见测试环境上的泛化能力。在下层，由上层塑造的奖励函数引导的RL智能体在具有可访问轨迹数据的有限训练环境集上学习策略；在上层，CMA-ES优化奖励塑造参数，以最大化在无法访问轨迹的独立验证环境上的累积未塑造奖励。在连续控制任务上的结果表明，GERS在未见测试环境上的表现优于标准RL基线。GERS的性能与DR相当，尽管DR将GERS的训练环境和验证环境合并为一个单一的训练集并需要轨迹访问，而GERS无法访问验证轨迹。这些结果证实，GERS在受限的数据访问约束下有效增强了泛化能力。

## Abstract
Reinforcement learning (RL) often suffers from performance degradation when deployed in environments that differ from those encountered during training. Existing techniques such as domain randomization (DR) mitigate this, but require access to diverse training environments and full trajectory observability, assumptions that fail in privacy-preserving or restricted scenarios where only scalar performance metrics are available. We propose Generalization via Evolutionary Reward Shaping (GERS), a bilevel optimization approach to improve generalization on unseen test environments using only scalar feedback from validation environments. At the lower level, an RL agent guided via a reward function shaped by the upper level learns a policy on a limited set of training environments with accessible trajectory data; at the upper level, CMA-ES optimizes the reward shaping parameters to maximize the cumulative unshaped reward on separate validation environments for which trajectory access is unavailable. Results on continuous control tasks indicate that GERS outperforms the standard RL baseline on unseen test environments. GERS performance is comparable to DR, despite DR treating the combined set of training and validation environments of GERS as a single training set that requires trajectory access, whereas GERS cannot access validation trajectories. These results confirm that GERS effectively enhances generalization under restricted data access constraints.

## 论文提出的方法论

### 核心思想
采用**双层优化**框架：
- **上层（奖励塑形优化）**：使用CMA-ES（协方差矩阵自适应进化策略）优化奖励塑形参数ν，目标最大化在验证环境上的累积未塑造回报J_val(π_θ*(ν))。
- **下层（策略学习）**：使用PPO在固定训练环境上，基于上层给出的塑形奖励函数ˆr_ν(s,a) = r(s,a) + ⟨ν, φ(s,a)⟩ 学习策略π_θ。其中φ(s,a)为状态-动作拼接特征。

### 关键技术细节
1. **线性奖励塑形**：塑形奖励为原始奖励加上特征线性组合，保持简单可调。
2. **CMA-ES**：作为黑箱优化器，仅需验证标量回报，适合无法计算梯度的场景；且支持并行评价。
3. **策略继承**：每次下层PPO训练前，使用历史最优策略参数初始化，降低计算成本。
4. **并行化**：CMA-ES每代生成λ个候选解，每个独立运行PPO（有限步数K）后评估验证分数。

### 算法流程（文字说明）
1. 预训练：用原始奖励在训练环境上运行PPO得到初始策略θ^(1)。
2. 对每一代g：
   - 从CMA-ES采样λ个候选ν_i。
   - 并行：每个候选初始化PPO为θ^(g)，用塑形奖励ˆr_{ν_i}训练K步，得到策略θ_i^(g)。
   - 在验证环境上评估每个策略的未塑形累积回报，得到适应度。
   - 更新CMA-ES分布。
   - 选择适应度最高的策略作为下一代的初始策略θ^(g+1)。
3. 输出最终策略和最佳奖励参数。

## 实验设计

- **任务与场景**：MuJoCo v5连续控制任务：InvertedPendulum, Hopper, HalfCheetah, Ant。每个任务修改两个物理参数（质量、摩擦力或重力），用系数c∈[0.1,3.0]缩放，生成不同环境。
- **数据划分**：
  - 训练环境：单个默认环境（c=1.0）。
  - 验证环境：25个均匀采样的c值，固定用于上层优化，仅提供标量回报。
  - 测试环境：100个从同一分布独立采样的c值，用于最终评价。
- **对比方法**：
  - BasePPO：在单个训练环境上训练3×10^7步。
  - DR（域随机化）：使用26个环境（1训练+25验证）作为训练集，需要完整轨迹，训练5×10^7步。
- **评价指标**：测试环境上的平均累积回报（4个episode/环境）。

## 论文的主要结论与发现

- GERS在所有四个任务上的测试性能均显著高于BasePPO，表明其有效改善泛化。
- GERS与需要轨迹访问的DR性能相当，甚至在某些任务上略优（Hopper, InvertedPendulum），证明在数据约束下GERS是一种有效替代。
- 验证集性能与测试集性能趋势一致，未发生过拟合到验证集的现象。
- 在单一参数超出分布范围（OOD）时鲁棒性良好，但双参数同时极端变化时性能下降。

## 优点

- **创新性**：首次将双层奖励塑形与进化优化结合，解决仅标量反馈下的泛化问题，适用于隐私/受限场景。
- **现实意义**：类似于自动化红队测试等实际应用（客户端黑盒验证）。
- **方法优雅**：利用CMA-ES并行性，配合策略继承减少计算开销。
- **公平比较**：对比DR时，严格保证DR使用更多数据（轨迹+更多环境），突出GERS在信息受限下的竞争力。

## 不足与局限

- **实验覆盖有限**：仅测试了密集奖励的连续控制任务，未涉及稀疏奖励、离散动作或视觉任务。
- **计算成本高昂**：需要大量下游PPO训练（每代λ个），限制了独立试验次数（仅5次），统计显著性不足。
- **上层收敛问题**：CMA-ES因下层训练不充分导致评价噪声，无法完全收敛（见附录A）。
- **消融不充分**：未单独验证奖励塑形 vs. 多策略选择（best-of-many）各自的贡献。
- **应用限制**：假设验证环境可以从相同分布采样，且上下层策略网络结构固定；实际部署可能需要额外调整。
