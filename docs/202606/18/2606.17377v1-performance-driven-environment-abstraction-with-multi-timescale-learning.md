## 摘要
我们研究了大型马尔可夫决策过程中用于决策的性能驱动环境抽象。与保留几何或拓扑结构不同，我们寻求直接优化决策质量的抽象。我们将抽象建模为通过聚合状态空间并在每个聚合状态内强制共享动作分布而获得的受控近似。对于固定划分，我们建立了一个性能保证，将值函数逼近误差与动作共享引入的损失分离开来。在此分析的指导下，我们开发了一个多时间尺度强化学习框架，该框架联合调整策略和树状结构的环境抽象。由此产生的算法基于Q值差异细化和粗化状态空间的区域，在性能与抽象大小和复杂性之间取得平衡。实验结果表明，与演员-评论家基线相比，该方法实现了显著的状态压缩、更高的样本效率和更快的重新规划能力。

## Abstract
We study performance-driven environment abstraction for decision-making in large Markov decision processes. Rather than preserving geometric or topological structure, we seek abstractions that directly optimize decision quality. We model abstraction as a controlled approximation obtained by aggregating the state space and enforcing a shared action distribution within each aggregated state. For a fixed partition, we establish a performance guarantee that separates value-function approximation error from the loss introduced by action sharing. Guided by this analysis, we develop a multi-timescale reinforcement learning framework that jointly adapts the policy and a tree-structured environment abstraction. The resulting algorithm refines and coarsens regions of the state space based on Q-value discrepancies, balancing performance against abstraction size and complexity. Empirical results demonstrate substantial state compression, improved sample efficiency, and faster replanning compared to actor-critic baselines.

#### 方法论：核心思想、关键技术细节与算法流程

论文的核心思想是将环境抽象视为一个受控的、状态空间的**聚合过程**，并通过多时间尺度学习框架联合优化策略和抽象结构。

1.  **核心建模**：
    *   **状态聚合 (State Aggregation)**: 将原始状态空间 $S$ 划分为不相交的“超状态” $\Gamma = \{ \gamma_1, \ldots, \gamma_m \}$。决策在抽象的 $\Gamma$ 上进行。
    *   **共享动作分布约束 (Same-Action-Distribution, SAD)**: 同一个超状态（$\gamma$）内的所有原始状态必须**共享同一个动作分布** $\pi_\Gamma(a|\gamma)$。这是引入性能损失的关键近似来源。

2.  **理论分析**：
    *   **性能保证 (Theorem 2)**：对于一个固定的抽象 $\Gamma$，论文分离并量化了聚合引入的性能损失，即最优值函数 $V^*$ 与聚合最优值函数 $\bar{V}^*_{\Gamma,\mu^*}$ 之间的差别。误差由两部分构成：
        *   $\epsilon_\Gamma$: **价值函数逼近误差**，源于用常数近似超状态内的价值函数变化。
        *   $\delta_\Gamma$: **动作共享误差**，源于强制同一超状态内的所有状态采用相同的动作，这限制了单个状态选择最优动作的能力。
    *   该分析为自适应状态聚合提供了理论依据：聚合的边界应该设置在动作选择差异较大的区域。

3.  **多时间尺度学习框架 (Algorithm)**:
    *   论文设计了一个联合学习策略和树结构抽象的多时间尺度算法。算法包含**四个**不同的时间尺度，从快到慢依次为：
        *   **快时间尺度 1: 聚合型演员-评论家 (Aggregated Actor-Critic)**：在给定的抽象 $\Gamma$ 上，使用标准的AC算法学习最优聚合策略。
        *   **中等时间尺度 2: Q值预估更新**：维护节点（超状态）和其潜在子节点的Q值估计（$\bar{Q}^{est}$），用来评估细化或粗化抽象带来的潜在收益。
        *   **慢时间尺度 3: 树结构更新**：基于Q值差异，定期执行扩展（Expand）和收缩（Collapse）操作，动态调整抽象树的结构。
    *   **树操作**：使用**四叉树（Quadtree）** 结构表示抽象。
        *   **扩展 (Expand)**: 当一个叶节点 $\gamma$ 的Q值差异（父节点与其子节点的最佳平均Q值之差）超过一个阈值 $3(1-\beta)\lambda$ 时，表明细化该区域能带来显著的性能提升，于是将该节点分裂。
        *   **收缩 (Collapse)**: 当上述差异小于该阈值时，说明细化的收益不大，为降低复杂度，将一组子节点合并回其父节点。
    *   **联合优化问题**：最终的学习目标平衡了性能与复杂度：$\min_T \frac{\bar{\epsilon}_T}{1-\beta} + \lambda|V_{leaf}(T)|$。
