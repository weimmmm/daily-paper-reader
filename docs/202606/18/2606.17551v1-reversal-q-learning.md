## 摘要
迭代生成建模技术（如流匹配）为建模复杂行为提供了强大工具，用于有效的离线强化学习。本文提出一种新的离策略强化学习算法，该算法基于先验数据训练流策略。我们的想法始于“扩展”马尔可夫决策过程框架，该框架将单个流细化步骤视为MDP中的独立动作。为了在此框架内实现离策略强化学习，我们应用了两种技术：通过“逆向”流生成虚拟的在策略轨迹，使该框架与先验数据兼容；并应用偏差-方差缩减技术来缓解离策略强化学习中的horizon诅咒。我们将由此产生的算法称为逆向Q学习（RQL）。RQL相较于先前基于流的强化学习方法具有若干优势：它无需通过时间反向传播，更好地利用了学到的价值函数，并直接训练完整、具有表现力的流策略。通过在50个具有挑战性的模拟机器人任务上的实验，我们表明，与最先进的基于流的离线强化学习算法相比，RQL实现了最佳的平均离线强化学习性能。

## Abstract
Iterative generative modeling techniques, such as flow matching, provide powerful tools to model complex behaviors for effective offline reinforcement learning (RL). In this work, we propose a new off-policy RL algorithm that trains a flow policy based on prior data. Our idea starts from the "expanded" Markov decision process (MDP) framework, which treats individual flow refinement steps as separate actions in an MDP. To enable off-policy RL within this framework, we apply two techniques: we generate virtual on-policy trajectories (by "reversing" flows) to make this framework compatible with prior data, and we apply a bias-and-variance reduction technique to mitigate the curse of horizon in off-policy RL. We call the resulting algorithm Reversal Q-learning (RQL). RQL has several advantages over previous flow-based RL methods: it does not suffer from backpropagation through time, makes better use of the learned value function, and directly trains the full, expressive flow policy. Through our experiments on 50 challenging simulated robotic tasks, we show that RQL leads to the best average offline RL performance compared to state-of-the-art flow-based offline RL algorithms.

## 2. 论文提出的方法论

- **核心思想**：基于“扩展 MDP”框架，将流策略的每个欧拉积分步骤（共 F 步）视为 MDP 中的一个独立动作，从而将原 MDP 的 horizon 扩大 F 倍。为了适用于离策略 RL，利用流模型确定性可逆的特性，从离线数据中的 (state, action) 对通过**逆向流**生成虚拟的完整流轨迹，再应用**多步回报**（multi-step returns）来缩短有效 horizon，使价值学习更稳定。
- **关键技术细节**：
  - **扩展 MDP 定义**：状态空间扩展为 eS = S × R^d × {0,...,F-1}，其中 d 是动作维度。F 步内只更新部分动作，每 F 步才与环境交互并获取奖励。折扣因子 γ 仅在最后一步应用。
  - **虚拟轨迹生成**：对于数据集中的每个 (s, a, r, s')，使用当前流策略的逆 ODE：`df/dθ = -v(s, θ, f)`，从 a 反向积分得到 `x_0, x_1, ..., x_{F-1}`，从而构造完整的扩展 MDP 轨迹。
  - **价值学习**：采用 **expectile regression**（类似 IQL）近似贝尔曼最优算子，损失函数为：
    ```
    L(V) = E[ ℓ_κ^2( V(s, x_f, f) - (r + γ V(s', x'_0, 0)) ) ]
    ```
    其中 κ > 0.5 用于近似 max 操作。
  - **策略学习**：采用 DDPG 风格的行为正则化损失：
    ```
    L(v) = -E[ V(s, x_f + v(s, x_f, f), f+1) ] + α L_BC(v)
    ```
    其中 L_BC 是标准的流匹配行为克隆损失，α 控制正则化强度。
  - **有效 horizon 缩减**：通过直接使用多步回报（从 f 跳到 f=F 的最终奖励）而非逐步 TD 更新，将有效 TD horizon 从 T×F 降低至 T，且由于虚拟轨迹是确定性且在策略的，该多步回报无偏且零方差。
- **算法流程**（文字描述）：
  1. 初始化价值函数 V 和流策略 v。
  2. 每次迭代：
     - 从数据集采样 (s, a, r, s')。
     - 通过逆向流计算 x_0～x_{F-1}，构建扩展 MDP 轨迹。
     - 用 expectile 损失更新 V。
     - 用行为正则化损失更新 v。

- **RQL 在 50 个任务上的平均性能最优**，尤其在长 horizon 高难度任务（如 cube-quadruple、humanoidmaze-large、puzzle-4x4）上显著优于所有基线。

## 7. 优点

- **避免 BPTT 不稳定**：通过扩展 MDP 将迭代过程解耦，无需通过流生成链反向传播。
- **高效利用价值函数**：策略损失直接使用一阶梯度信息，比仅用零阶信息（如加权回归）更有效。
- **无偏零方差多步回报**：利用流可逆性生成确定性在策略轨迹，使多步 TD 目标无偏且零方差，这是普通离策略 RL 无法做到的。
- **直接训练完整流策略**：无需蒸馏或分层结构，保持策略表达力。
- **在多个复杂任务上达到 SOTA**，尤其擅长长 horizon 和高维控制。

## 8. 不足与局限

- **超参数敏感**：α（BC 正则化）和 κ（expectile）需要针对每类任务仔细搜索，影响实用便利性。
- **实验未覆盖其他领域**：仅验证于机器人控制，未扩展到图像生成、自然语言等其他生成建模应用。
- **无算力报告**：缺少 GPU 型号、数量、训练时间等细节，不利于复现和比较效率。

- **假设流确定性可逆**：若使用随机扩散模型则无法直接应用逆向流方法，限制了泛化性。
