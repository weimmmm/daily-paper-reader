在本报告中，我们介绍了Hy-Embodied-0.5-VLA，缩写为HyVLA-0.5，这是一个覆盖完整机器人学习栈的端到端系统：数据收集、模型设计、持续预训练和监督微调、强化学习后训练以及现实世界部署。该栈中的每个组件都扮演着独特的角色。

## Abstract
In this report, we present Hy-Embodied-0.5-VLA, abbreviated as HyVLA-0.5, an end-to-end system that spans the full robot learning stack: data collection, model design, continued pre-training and supervised fine-tuning, RL post-training, and real-world deployment. Each component serves a distinct role in this stack.

### 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：当前 Vision-Language-Action (VLA) 模型的研究多集中在模型架构本身，而忽视了从数据收集到真实部署的完整工程栈。这导致了三个关键瓶颈：
    - **数据端**：传统遥操作缺乏触觉反馈，模仿人类数据精度不足，且存在跨体表现在（embodiment gap）、控制差距（control gap）和感知差距（perception gap）。
    - **模型与训练端**：离散化动作表示限制控制频率和精度；标准模仿学习难以达到灵巧操作；强化学习依赖易碎的奖励模型。
    - **部署端**：高容量 VLA 模型推理速度慢于机器人伺服频率，且分块动作存在边界不连续问题，部署很少被视为首要设计目标。
- **整体含义**：论文提出 HyVLA-0.5，将机器人学习从孤立的模型研究提升为统一的、可落地栈（pipeline）的系统级方案，旨在填补从研究到实际机器人操作之间的工程鸿沟。它是**端到端**的，涵盖了数据收集、模型架构、多阶段训练（预训练 + SFT + RL）、以及真实环境部署的所有环节。

### 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：将 VLA 模型的开发视为一个完整栈，每个环节（数据、模型、训练、部署）协同设计，而非单独优化。通过**高精度人机数据**、**模块化解耦的架构**、**无奖励模型的偏好优化**以及**异步平滑执行**，实现可部署的通用机器人。
- **关键技术细节**：

    - **高精度数据收集（Hy-UMI-10K）**：
        - 设计 **指尖式 UMI 设备** + **外置光学动捕笼**，替代传统手持设备（SLAM 精度低、无法传递指尖力）和遥操作。
        - 获取超过 10,000 小时的**自我中心（egocentric）**、**亚毫米精度**的人类演示，覆盖 70 种任务。
        - 指尖结构和六维力传感器恢复自然触觉，动捕提供全局高保真轨迹，可直接用于后训练（RL）。
    - **模型架构**：
        - **基座**：Hy-Embodied-0.5-MoT（4B参数，Mixture-of-Transformers），具备模态自适应计算和原生分辨率图像编码。延续此基座的空间先验和快速微调能力。
        - **动作专家（Action Expert）**：采用**流匹配（Flow Matching）** 模型直接预测连续动作速度场，而非离散化动作 token。支持高频、连续控制。
        - **紧凑记忆编码器（Compact Memory Encoder）**：通过**交错时-空注意力**（temporal-spatial attention）将 K 帧历史图像压缩为当前帧表征，不增加传递给 VLM 的 token 数量。参数可复用预训练 ViT。
        - **Delta-Chunk 动作表示**：预测相对于当前末端执行器位姿的相对增量动作块（delta-chunk）。这解耦了策略学习与特定的机器人运动学（kinematics），便于跨体表迁移。
    - **训练与微调**：
        - **阶段一：持续预训练**：在 Hy-UMI-10K 数据集上预训练 VLA 模型，学习通用动作先验。使用流匹配损失。
        - **阶段二：监督微调（SFT）**：分为两个轨道：
            - **Track-A（同体适应）**：在同一机器人平台上通过遥操作收集数据，微调模型并部署回同一平台。
            - **Track-B（跨体迁移）**：仅使用任务相关的 UMI 演示数据，微调后部署到形态不同的目标机器人（如 JAKA K1、Astribot S1），无需目标机器人的遥操作数据。
    - **强化学习后训练（FlowPRO）**：
        - 提出 **Proximalized Preference Optimization (PRO)-based 离线 RL 算法**。
        - **数据收集**：通过**人为干预-回滚**机制，自动生成成对的成功/失败轨迹（τ_w, τ_l），无需训练奖励模型。
        - **损失函数 RPRO**：结合对比项（L_con 拉近成功动作、推远失败动作）和近端正则化项（L_reg），防止奖励过度膨胀（reward hacking）。同时保留监督项 L_SFT。
        - **特性**：对比度梯度抵消（当 a_w = a_l 时 L_con 为零），允许安全地将 SFT 数据通过同一损失函数混合训练。
    - **部署**：
        - **平台映射（Platform Mapper）**：将 delta-chunk 动作通过当前末端位姿映射到目标机器人的世界坐标系或底盘坐标系，保持学习到的动作接口不变。
        - **异步推理（Asynchronous Execution）**：将推理线程和执行线程解耦，通过线程安全缓冲区重叠前向传播与伺服执行，隐藏推理延迟。
        - **贝塞尔块平滑（Bézier Chunk Stitching）**：使用三次贝塞尔曲线在过时块与新块之间生成 C1 连续（位置、速度连续）的连接，消除块间不连续性。通过历史方向和未来方向估计切线方向，无需额外学习。

### 3. 实验设计

- **数据集**：

    - 真实世界：π₀、π₀.₅（同等参数量、数据量和迭代次数训练）。
    - RL 后训练：DAgger、π₀.₆*（advantage-conditioned regression）。

### 6. 论文的主要结论与发现

- **模拟效果**：HyVLA-0.5 在 RoboTwin 2.0 上以 90.9%（Clean）和 90.1%（Randomized）的平均成功率取得最优，显著超越 π₀/π₀.₅，略优于 JoyAI-RA。消融表明：紧凑记忆编码器带来 2.1-1.5 个点增益，UMI 预训练再带来 0.7 个点增益。
- **真实世界效果**：
    - **Track-A（同体）**：UMI 预训练显著提升了精密操作（如折叠眼镜、拉拉链）的成功率，尤其是在需要亚厘米精度的关键步骤，验证了高精度数据预训练的价值。
    - **Track-B（跨体）**：仅用 UMI 演示微调的 HyVLA-0.5 在 JAKA K1 和 Astribot S1 上大幅领先 π₀/π₀.₅，证明大规模 UMI 预训练提供了强的**体表无关动作先验**，使少量 UMI 数据即可实现可部署的跨体迁移。
    - **力模态验证**：通过指尖力 / 扭矩信号，策略成功在 Unitree G1 上区分轻重盒子并放置，表明数据中的触觉信息可用于学习力感知操作。
- **RL 后训练**：FlowPRO 在所有四个任务上（Bottle, Cap, USB, Zip）均获得最高成功率（98%-99%）和最短完成时间，显著优于 DAgger 和 π₀.₆*。
- **核心贡献**：论文证实了从高精度数据收集、体表解耦表示、无奖励偏好优化到异步平滑部署的完整栈协同工作是实现可部署通用机器人的可行路径。

### 7. 优点

- **系统完整性**：不是单一模型，而是覆盖数据、模型、训练、部署的端到端栈，每个环节都针对实际硬件约束设计。
- **数据创新**：指尖 UMI + 动捕提供亚毫米精度且包含触觉的数据，既可预训练又可作为 RL 数据，复用性强。
- **体表解耦**：Delta-chunk 表示和紧凑记忆编码器使策略学习与机器人运动学无关，零精度损失实现跨体迁移。
- **无奖励 RL**：FlowPRO 无需奖励模型、价值网络，仅通过干预-回滚生成偏好对，解决了接触式操作中奖励设计难的痛点。
- **部署实用**：异步推理 + 贝塞尔平滑是训练无关的轻量级解决方案，保证了实际闭环控制的高频和流畅性。
- **实验严谨**：模拟、真实、同体、跨体、消融、力觉等多维度评估，结果统计可靠。
