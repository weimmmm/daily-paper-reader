## 摘要
人形机器人已具备强大的运动能力，但在多变地形上的可靠导航仍具挑战性，因为避障必须与动态可行的运动相协调。在本工作中，我们提出了GuideWalk，一个统一的端到端框架，将可通行性感知的导航引导与地形自适应运动教师相结合，用于人形机器人导航。具体而言，我们引入了一个导航模块，提供显式的速度引导，将避障与地形条件解耦，从而在不同环境中实现稳健的规划。我们提出了一种复合教师蒸馏方案，将目标导向指令和动态一致的动作聚合并蒸馏到单一策略中。为进一步提升鲁棒性，蒸馏策略通过强化学习和辅助行为克隆目标进行精炼，在保留理想教师行为的同时促进探索。实验表明，GuideWalk在保持稳定人形机器人运动的同时，实现了稳定有效的导航。

## Abstract
Humanoid robots have achieved strong locomotion capabilities, but reliable navigation on versatile terrains remains challenging because obstacle avoidance must be coordinated with dynamically feasible motion. In this work, we present GuideWalk, a unified end-to-end framework that integrates traversability-aware navigation guidance with terrain-adaptive locomotion teacher for humanoid navigation. Specifically, we introduce a navigation module that provides explicit velocity guidance, decoupling obstacle avoidance from terrain conditions to enable robust planning across diverse environments. We propose a composite teacher distillation scheme, where goal-directed commands and dynamically consistent actions are aggregated and distilled into a single policy. To further improve robustness, the distilled policy is refined with reinforcement learning and an auxiliary behavior cloning objective, which promotes exploration while preserving desirable teacher behaviors. Experiments demonstrate that GuideWalk achieves stable and effective navigation while maintaining stable humanoid locomotion.

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
- 采用**两阶段训练流水线**：第一阶段通过DAgger（数据集聚合）从复合教师蒸馏学生策略；第二阶段使用PPO强化学习联合辅助行为克隆（BC）损失进行精炼。
- **复合教师**由两个模块组成：
  - **导航引导**：基于DWA（动态窗口法）的局部规划器，提供显式速度指令（前向线速度v和偏航角速度ω），解耦避障与地形条件。
  - **运动教师**：预训练的受特权观测的locomotion策略（通过AMP和MoCap数据训练），由导航指令驱动生成关节动作。

### 关键技术细节
1. **导航引导（Navigation Guidance）**：
   - 输入：机器人当前位姿、目标位置、FoV内障碍观测（匹配深度相机规格）。
   - 采用差动驱动运动学模型（非全向），因为locomotion教师对横向速度跟踪鲁棒性较差。
   - 优化目标：$G(v, \omega) = \sigma(\alpha \cdot \text{heading} + \beta \cdot \text{dist} + \gamma \cdot \text{velocity})$，受限于速度空间约束（运动学极限、动态窗口、避碰）。
   - 近似通过网格搜索求解最优速度对$(v^*, \omega^*)$。

2. **运动教师（Locomotion Teacher）**：
   - 参数化策略$\pi_{lt}: \mathcal{V} \times \tilde{\mathcal{O}} \to \mathcal{A}$，其中$\tilde{\mathcal{O}}$包含特权信息（如真实地形高程图、无噪声本体感知等）。
   - 通过RL+AMP（对抗运动先验）训练，参考行为来自自采集动作捕捉数据，促进类人运动，减少手工奖励设计。

3. **复合教师蒸馏（DAgger Distillation）**：
   - 学生策略$\pi_s$在部分可观测下运行（标准观测：深度图像、高程图、本体感知等）。
   - 学生与环境交互收集状态轨迹，查询冻结的复合教师得到动作，存入数据集。优化目标：最小化学生与教师动作的MSE。

4. **RL精炼（RL Finetune）**：
   - 采用PPO最大化累积回报，其中奖励函数包含DWA速度跟踪项（连接导航与low-level控制）、位置跟踪、航向跟踪、姿态惩罚、敏捷速度奖励、停滞惩罚等。
   - 联合优化：$L = L_{\text{PPO}} + \lambda L_{\text{BC}}$，其中$L_{\text{BC}}$为学生与教师动作的MSE，$\lambda$为平衡权重（实验中设为0.2）。

### 公式/算法流程（文字说明）
- 阶段1：学生策略$\pi_s$通过DAgger迭代训练，收集状态-动作对$(o, \tilde{o}, a^*)$，优化$\min \mathbb{E}[\|\pi_s(o) - \pi_t(\tilde{o})\|^2]$。
- 阶段2：固定$\pi_s$初始化，用PPO和辅助BC损失进一步优化：$\min \mathbb{E}[-\text{PPO loss} + \lambda \|\pi_s(o) - \pi_t(\tilde{o})\|^2]$。

- GuideWalk在仿真中实现了高导航成功率（99.0% NSR）和绝大多数地形的高穿越成功率（>96% TSR），同时具有较低的碰撞风险（PO=0.65m）和较短的导航时间（15.51s）。

## 7. 优点：方法或实验设计上的亮点
