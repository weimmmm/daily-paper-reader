## 摘要
深度强化学习在社交导航领域展现出潜力，但其实际部署仍受到持续的仿真到现实差距的阻碍，这一差距源于简化的一阶动力学和特定上下文的人类状态估计流程。本文提出一个统一框架，以解决这些限制，从而生成适用于现实部署的动态可行导航策略。首先，理论分析表明，模拟与实际机器人位置之间的跟踪误差随着控制阶数的增加呈指数衰减，这促使使用高阶控制输入作为深度强化学习的动作空间。针对差速驱动机器人，开发了一种二阶控制公式，并辅以随机迭代线性二次型调节器，该调节器通过散度最小化目标对策略进行预训练。其次，为了避免相机-激光雷达融合带来的额外系统复杂性，引入了一种仅使用二维激光雷达的基于聚类的人体跟踪流程。根据空间邻近性和速度相似性关联人体检测，从而可靠地区分附近行人，并通过时间聚合获得稳定的速度估计。第三，我们引入了一个无偏残差门控模块，以平衡基于反应和基于记忆的行为，同时处理时变的人群规模，这两点对社交导航至关重要。最终策略KinematicRL持续提升运动学性能，并适应不同数量的检测到的人。在真实环境中的实验表明，当与所提出的跟踪流程结合时，KinematicRL可以在几乎无需修改的情况下部署在真实的差速驱动机器人上。

## Abstract
Deep Reinforcement Learning (DRL) has shown promise for social navigation, yet its real-world deployment remains hindered by a persistent sim-to-real gap arising from simplified first-order dynamics and context-specific human state estimation pipelines. This work presents a unified framework that addresses these limitations to produce dynamically feasible navigation policies suitable for real-world deployment. First, theoretical analysis reveals that tracking error between simulated and actual robot position decays exponentially with increased control order, motivating the use of higher-order control inputs as DRL action space. A second-order control formulation tailored to differential drive robots is developed, complemented by a stochastic iterative Linear Quadratic Regulator (iLQR) that pretrains the policy via a divergence minimization objective. Second, to avoid the added system complexity of camera-LiDAR fusion, a cluster-based human tracking pipeline using only 2D LiDAR is introduced. Human detections are associated according to both spatial proximity and velocity similarity, enabling reliable differentiation of nearby pedestrians and yielding stable velocity estimates through temporal aggregation. Third, we introduce an unbiased residual gating block to balance reaction- and memory-based behaviors while handling time-varying crowd sizes, both critical for social navigation. The resulting policy, KinematicRL, consistently improves kinematic performance and adapts to varying number of detected humans. Experiments in real-world environments demonstrate that, when combined with the proposed tracking pipeline, KinematicRL can be deployed on a real differential drive robot with minimal modifications.

### 1. 论文的核心问题与整体含义（研究动机和背景）

- **背景**：深度强化学习（DRL）在社交导航（crowded social navigation）中表现出潜力，但实际部署受限于**仿真到真实（Sim-to-Real）差距**。传统DRL方法多在仿真中使用**简化的一阶动力学模型**（假设机器人能瞬间跟踪指令速度）和**理想化的智能体状态**，而真实机器人受限于执行器延迟和加速度约束，且人类状态必须从传感器估计。
- **核心问题**：如何同时解决三个关键障碍：
  1. 一阶动作空间导致跟踪误差大，缺乏动力学可行性；
  2. 常用的人类状态估计依赖相机-激光雷达融合，增加系统复杂度与标定漂移；
  3. 人群数量随时间变化（遮挡/传感器限制），现有Transformer在变长输入上训练不稳定。
- **整体目标**：提出一个统一框架**KinematicRL**，在仿真中保证动力学可行性，仅用2D LiDAR实现鲁棒跟踪，并通过门控架构适应动态人群，从而无缝迁移到真实差速驱动机器人。

### 2. 论文提出的方法论

#### 2.1 核心思想
- 使用**高阶控制（二阶）** 作为DRL动作空间，从根本上降低跟踪误差；通过**随机iLQR**预训练二阶策略，再通过PPO微调。
- 仅依靠**2D LiDAR**进行基于聚类的行人跟踪，利用空间-速度相似性数据关联。
- 设计**无偏残差门控Transformer**，平衡反应式与记忆式行为，并稳定处理变长输入。

#### 2.2 关键技术细节
- **高阶控制（Theorem 1）**：理论证明跟踪误差与\((\Delta t)^{n-k}/(n-k)!\)成正比，控制阶数\(n\)越高误差指数级减小。实际采用**二阶控制**：状态\([x,y,\theta,v,\omega]\)，动作\([a,\alpha]\)（线加速度和角加速度），自然满足加速度约束。
- **随机iLQR预训练**：将社交导航视为最大熵控制问题，求解得到高斯策略\(\pi(u_t|x_t)=\mathcal{N}(\hat{u}_t+K_t\delta x_t+k_t, Q_{uu,t}^{-1})\)。通过**前向KL散度最小化**（M-投影）从iLQR演示中初始化策略，结合DAgger减少分布偏移。代价函数包含目标到达、光滑性、碰撞避免（非凸项通过Hessian正则化处理）。
- **聚类跟踪管道**：
  - 利用**Li2Former**检测人体中心点；
  - 数据关联使用**空间-速度加权距离**\(J=(1-\alpha)\|\hat{p}_n - p_{\text{cluster}}\| + \alpha \|\hat{v}_n - v_{\text{cluster}}\|\)，匈牙利算法分配；
  - 每个聚类维护历史缓冲区，通过**加权历史平均**（时间衰减权重）估计速度，卡尔曼滤波去噪。
- **无偏残差门控Transformer**：
  - 标准Transformer在RL中初始化不稳定（LayerNorm破坏Markov性），且残留连接固定融合反应/记忆路径。
  - 采用**前LayerNorm**、**GRU门控**（可学习开关，\(\Gamma_u\)控制注意力流贡献）、**修改的残差连接**（第二个GRU直接从输入embeding跳接），避免信号随深度衰减。
  - 双编码器结构：**空间编码器**（Masked MHSA处理当前人群）→ **时间编码器**（处理多步历史）。
  - **场景基域随机化**：改变人类起点-终点配置、速度随机化，增加状态空间熵，迫使策略学习主动避让而非被动等待。

#### 2.3 算法流程（文字说明）
1. 使用随机iLQR在仿真环境中生成\(n_{\text{ilqr}}\)轮演示轨迹（含状态-动作对）。
2. 通过前向KL散度最小化（M-投影）从演示中学习初始策略\(\pi_{\text{bc}}\)。
3. 用DAgger执行\(n_{\text{dagger}}\)轮：策略\(\pi_{\text{bc}}\)采取动作，iLQR提供专家动作，收集数据并重新拟合策略。
4. 使用PPO强化学习微调策略：奖励函数包含目标到达奖励、碰撞惩罚、社交距离惩罚（软约束）和时间惩罚。

### 6. 论文的主要结论与发现

- **动力学可行性关键**：二阶控制+随机iLQR预训练使策略输出天然满足加速度约束，避免“在仿真中使用不可行动作钻空子”的Sim-to-Real gap（SARL在改为二阶时成功率从0.99暴跌至0.52）。
- **跟踪管道有效**：仅用2D LiDAR实现稳定速度估计（历史加权平均误差最低0.11 m/s），空间-速度关联显著减少身份交换。
- **门控Transformer优于普通Transformer**：无偏残差门控+场景域随机化使策略学会主动避让，在变长人群下泛化良好（训练时4-6人，测试9人仍优于SARL）。
- **真实部署可行**：在真实机器人上实现10Hz控制频率，轨迹平滑，主动绕行行人。

### 7. 优点

1. **理论支撑充分**：Theorem 1严格证明高阶控制降低跟踪误差，为动作空间设计提供数学基础。
2. **工程实用性强**：仅依赖2D LiDAR单传感器，避免相机标定和融合复杂性；网络轻量（ARM上运行10Hz），适合边缘部署。
3. **架构创新有效**：无偏残差门控解决变长输入训练不稳定问题，是Transformer在RL社交导航中的成功应用。
4. **训练策略巧妙**：场景域随机化（不仅是鲁棒性工具，更是注意力学习的必要条件）和M-投影模仿学习加速收敛。
5. **开源友好**：提供ROS节点、代码和视频，便于复现和扩展。

### 8. 不足与局限

1. **碰撞率仍较高**：在高度动态场景下KinematicRL碰撞率0.15，高于SARL的0.01，可能因二阶非完整模型灵活性降低。作者指出需引入显式安全保证（如CBF）。
2. **感知模块计算负担**：Li2Former依赖GPU（RTX 2060/Orin Nano），机器人本机无法运行；若使用低算力平台需模型量化或压缩。
3. **未完全解决静态障碍物**：论文只关注行人，未涉及静态障碍物（如墙壁、家具）的避让，实际部署需额外层。
4. **真实实验量少**：只有1个定性示例，缺乏多场景、多轮次统计结果，难以评估真实世界鲁棒性。
5. **训练时假设人类非合作**：实际场景中存在合作行人，策略可能过于保守或激进，未对此消融。
6. **控制频率依赖**：虽然控制率10Hz，但论文指出较高频率可减轻jerk近似误差，对不同机器人的通用性需进一步验证。
