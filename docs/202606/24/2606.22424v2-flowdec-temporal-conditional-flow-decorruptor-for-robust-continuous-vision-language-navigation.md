## 摘要
连续环境下的视觉与语言导航（VLN-CE）要求智能体在未见场景中遵循自然语言指令。尽管大型模型（LMs）推动了VLN-CE的发展，但其性能仍因真实世界中的视觉损坏而严重下降，这是一个关键但尚未充分探索的领域约束。我们提出了时间条件流去噪器（FlowDec），这是一种专为基于LM的VLN-CE设计的新型图像恢复框架。FlowDec整合了一种混合时间条件策略，以将生成流路径与历史上下文对齐，并采用动作质心引导滤波动态评估和整合输出。大量实验表明，FlowDec在导航精度和生成延迟方面均优于最先进的去噪方法。我们的方法为在不可预测的真实世界条件下实现鲁棒具身导航建立了一种高效且稳健的范式。

## Abstract
Vision-and-Language Navigation in Continuous Environments (VLN-CE) requires agents to follow natural-language instructions in unseen scenes. While Large Models (LMs) have advanced VLN-CE, their performance remains severely degraded by real-world visual corruptions, a critical yet underexplored domain constraint. We introduce Temporal Conditional Flow Decorruptor (FlowDec), a novel image restoration framework tailored for LM-based VLN-CE. FlowDec integrates a hybrid temporal conditioning strategy to align the generative flow path with historical context and employs action-centroid guided filtering to dynamically assess and integrate outputs. Extensive experiments demonstrate that FlowDec outperforms state-of-the-art decorruption methods in both navigation accuracy and generation latency. Our approach establishes a robust, efficient paradigm for resilient embodied navigation in unpredictable real-world conditions.

- **研究动机**：连续环境下的视觉语言导航（VLN-CE）要求智能体在未知场景中根据自然语言指令导航。尽管大型模型（LMs）在VLN-CE中取得了显著进展，但真实世界中的视觉损坏（如运动模糊、低光照、雨雪、镜头污染等）会导致性能严重下降，而这一问题在现有研究中尚未被充分探索。
- **核心挑战**：VLN-CE系统需要处理未知且多样化的损坏类型，保持帧间时间一致性，维持物理上有意义的动作预测，并满足实时性要求。现有图像恢复方法（如盲图像去噪BID、测试时自适应TTA）要么只关注感知质量，要么忽略时间一致性，要么计算开销过大。
- **整体贡献**：本文首次明确研究并解决VLN-CE中的图像去噪问题，提出FlowDec框架，在不修改导航骨干网络的前提下，提升对视觉损坏的鲁棒性，并保持实时性能。

## 2. 论文提出的方法论

### 2.1 核心思想
- 采用条件流匹配（Conditional Flow Matching, CFM）在潜在空间中生成去噪图像。
- 提出混合时间条件策略，将历史帧信息融入生成流路径，增强时间一致性。
- 引入动作质心（Action Centroids）指导的潜在滤波机制，动态评估并融合来自不同条件路径的输出。

### 2.2 关键技术细节

#### （1）条件流匹配基础
- 在潜在空间（通过预训练VAE编码器E得到）中进行流匹配，学习速度场vθ，使随机噪声z0演化到真实图像潜在z1。
- 损失函数为条件流匹配损失LCFM，通过最小化真实速度与预测速度的L2距离来优化。

#### （2）混合时间条件策略
- 训练时使用三种条件ci（i=1,2,3）：
  - c1：仅当前增强图像的潜在，表示单帧去噪。
  - c2：结合当前增强图像、前帧增强与真实图像的差值、以及动作标签，近似损坏特征。
  - c3：用前帧去噪结果替代真实图像，逐步弥合合成与真实域差距。
- 训练调度：c1权重固定，早期c2为主，后期逐渐增加c3比重。

#### （3）动作质心引导滤波（推理阶段）
- 在训练中为每个原子动作（前进、左转、右转等）计算潜在差分量的均值与方差，形成动作质心分布。
- 推理时，主路径使用c1生成，只有当当前帧与前一帧的潜在差与对应动作质心的马氏距离超过阈值wθ时，才调用c3辅助生成，并加权融合输出。
- 起始点改进：将随机噪声与损坏图像线性插值作为起始点z'0，加速推理并减少随机性。

### 2.3 公式与算法流程
- 训练阶段：构造动作质心 → 对每批数据应用增强 → 按调度选择条件ci → 计算LCFM损失并更新模型。
- 推理阶段：对每帧，用c1生成 → 若非首帧则计算马氏距离w → 若w > wθ则用c3生成并融合 → 解码后送入VLN模型。

## 3. 实验设计

### 3.1 数据集与场景

- **损坏类型**：12种（高斯噪声、散粒噪声、脉冲噪声、散焦模糊、运动模糊、雪、雾、光线不足、像素化、JPEG压缩、对比度、亮度），重点评估其中6种最破坏性的。
- **真实世界实验**：Unitree GO2四足机器人，4个任务（室内2个、室外2个），共80次试验。

### 3.2 评测指标
