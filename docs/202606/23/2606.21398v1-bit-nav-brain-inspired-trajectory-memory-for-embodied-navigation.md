## 摘要
用于具身导航的视觉-语言模型（VLM）依赖于从不断增长的轨迹历史中选择固定数量的帧。随着情节的延长，这种选择变得越来越稀疏，然而先前的研究表明，将帧数从8帧扩展到64帧时，准确率并未提升，这表明瓶颈不在于帧数量，而在于表征本身。稀疏的帧选择无法捕捉长时推理所需的结构化行为信号：转向模式、累积位移和路径拓扑。我们提出了BIT-Nav（面向导航的脑启发轨迹记忆），这是一种通过紧凑的学习轨迹记忆来增强冻结VLM导航管道的框架。受海马路径整合的启发（其中空间经验被压缩为结构化的情景痕迹，而非存储为原始感官重放），BIT-Nav在动作和相对姿态序列上训练一个双向GRU编码器，通过对具有相同行为意图的轨迹前缀进行多正例InfoNCE对比目标。得到的嵌入通过轻量级MLP投影到VLM令牌空间，并在每个决策步骤中作为单个记忆令牌注入，从而以恒定的令牌成本（与情节长度无关）将结构化的运动历史条件化到模型中。

## Abstract
Vision-Language Models (VLMs) for embodied navigation rely on selecting a fixed number of frames from a growing trajectory history. As episodes extend, this selection grows increasingly sparse, yet prior work shows no accuracy gain when scaling from 8 to 64 frames, suggesting the bottleneck is not frame quantity but the representation itself. Sparse frame selection cannot capture the structured behavioral signal that long-horizon reasoning requires: turning patterns, cumulative displacement, and path topology.   We introduce BIT-Nav (Brain-Inspired Trajectory Memory for Navigation), a framework that augments frozen VLM navigation pipelines with a compact learned trajectory memory. Motivated by hippocampal path integration, where spatial experience is compressed into structured episodic traces rather than stored as raw sensory replay, BIT-Nav trains a Bi-GRU encoder over action and relative pose sequences via a multi-positive InfoNCE contrastive objective on trajectory prefixes sharing the same behavioral intent. The resulting embedding is projected into the VLM token space via a lightweight MLP and injected as a single memory token at each decision step, conditioning the model on structured motion history at constant token cost regardless of episode length

## 1. 核心问题与整体含义（研究动机和背景）
- **问题**：现有视觉-语言模型（VLM）在具身导航中依赖固定数量的历史帧采样来提供上下文。随着导航片段延长，帧采样变得稀疏，丢失了中间运动结构（如转向模式、累积位移、路径拓扑），导致长时推理瓶颈。先前工作（NaVILA）表明，将帧数从8增加到64并不能提升导航准确率，说明瓶颈不在于帧数量，而在于表征形式本身。
- **背景**：生物导航系统（海马-内嗅皮层）并不存储原始感官回放，而是将空间经验压缩为结构化情景痕迹，支持路径整合。受此启发，论文提出将轨迹历史作为显式模态注入VLM，使模型能够在决策时利用紧凑、结构化的运动记忆。

## 2. 方法论：核心思想、关键技术细节
- **核心思想**：训练一个双向GRU（Bi-GRU）编码器，将动作和相对位姿序列编码为固定维度的向量，然后通过轻量级MLP投影器映射到VLM的令牌空间，在每个决策步骤作为单个记忆令牌注入冻结的VLM，使模型能够同时推理当前视觉观察、语言指令和结构化运动历史。
- **关键技术细节**：
  - **运动特征工程**：将原始离散动作（前进、左转、右转、停止）转化为7维特征向量：`[cx, cy, sinθ, cosθ, Δθ, is_forward, is_turn]`，包含累积平面位置、累积航向角、每一步航向变化量、前向/旋转指示符。步长0.25m，转角15°。
  - **对比学习训练**：基于SimCLR框架，使用**多正例InfoNCE损失**。训练数据由3600个起点-终点组构成，每组包含10条不同路径（轨迹增强），同一组内轨迹共享起点和终点但路径不同。每批次采样6组（共60条轨迹）。对每个锚点，正例为同组其他9条轨迹，负例为其余50条。损失函数为：
    - \( L_i = -\frac{1}{|P(i)|} \sum_{j\in P(i)} \log \frac{\exp(z_i^\top z_j / \tau)}{\sum_{k\neq i} \exp(z_i^\top z_k / \tau)} \)
  - **BITE块（Bi-GRU Trajectory Encoder）**：2层双向GRU，隐藏维度256，每个时间步输出512维，进行掩码均值池化得到512维向量，再经投影头降为128维，L2归一化后作为轨迹嵌入。
  - **VLM对齐**：冻结Bi-GRU编码器（302,208参数）和Qwen3-VL-8B（87.7亿参数），仅训练一个2层MLP投影器（约446万参数，<0.06%模型大小），将128维轨迹嵌入映射到4096维VLM令牌空间，作为单个令牌（`<gru>`）附加到当前输入序列之前：`X_t = [e_traj, e_visual, e_frames, e_instr]`。
- **算法流程**：训练阶段：从R2R/RxR提取轨迹→构造7D运动特征→分组对比学习→训练Bi-GRU编码器。推理阶段：实时编码前缀轨迹→投影为单令牌→注入冻结VLM→生成下一动作。

## 6. 主要结论与发现
- BIT-Nav能够以恒定计算代价（单令牌）提供结构化运动历史，克服帧稀疏问题。
- 在累积航向估计（Q1）上达到0.83准确率，远优于无历史基线的≈0.07和文本历史基线的≈0.07-0.10；在动作频率推断（Q2）上匹配文本历史基线，同时令牌消耗降低至4.5%（41 vs 903 tokens at step 124）。
- 定性实验显示BIT-Nav能正确报告转向序列和位置变化，而Base Qwen依赖当前帧产生历史错误描述。
- 实验中观察到**涌现的指令定位**能力：模型无需显式监督即可推断已完成的子指令和剩余步骤。

## 7. 优点
- **脑启发设计**：受海马路径整合启发，将空间经验压缩为结构化痕迹，生物合理性。
- **高效表征**：单令牌固定开销，与轨迹长度无关，解决长序列瓶颈。
- **模块化与冻结性**：编码器与VLM均冻结，仅训练轻量投影器，保持原有推理能力，易于集成现有VLM导航框架。
- **对比学习策略新颖**：多正例InfoNCE + 分组批量构造，无需负例挖掘，学习轨迹行为意图的等价类。
- **丰富的特征工程**：7D运动特征既保留局部动作语义又提供全局几何信息，避免将原始ID直接输入。

## 8. 不足与局限
