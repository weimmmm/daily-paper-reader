## 摘要
无监督强化学习旨在无需外在奖励的情况下预训练可扩展的、以技能为条件的策略，为下游控制任务提供基础。尽管最近取得了进展，我们认为当前的非策略无监督强化学习方法受到两个关键且被忽视的瓶颈的限制：（1）非平稳的技能语义和（2）脆弱的泛化能力。为了解决这些挑战，我们提出了GenDa（可泛化数据高效智能体），一个用于鲁棒无监督强化学习的统一框架。首先，我们引入了一种技能重标记机制，以缓解非平稳性并显著提高预训练的数据效率。其次，我们提出了互补信息瓶颈，鼓励学到的技能策略关注以自我为中心的特征，并对下游任务的分布偏移具有鲁棒性。通过多种实验，我们证明GenDa显著增强了无监督强化学习的可扩展性，具有优越的泛化能力和数据效率。我们的代码和视频可在https://ihatebroccoli.github.io/official-GenDa获取。

## Abstract
Unsupervised Reinforcement Learning (URL) aims to pre-train scalable, skill-conditioned policies without extrinsic rewards, serving as a foundation for downstream control tasks. Despite recent progress, we argue that current off-policy URL methods are limited by two critical, overlooked bottlenecks: (1) non-stationary skill semantics and (2) brittle generalization. To address these challenges, we propose GenDa (Generalizable Data-efficient Agent), a unified framework for robust unsupervised reinforcement learning. First, we introduce a skill relabeling mechanism to mitigate non-stationarity and significantly improve data efficiency for pre-training. Second, we propose a Complementary Information Bottleneck (CIB), encouraging the learned skill policy to focus on ego-centric features and become robust to distribution shifts for downstream tasks. Through various experiments, we demonstrate that GenDa significantly enhances the scalability of URL with superior generalizability and data efficiency. Our code and videos are available at https://ihatebroccoli.github.io/official-GenDa.

## 1. 核心问题与整体含义（研究动机和背景）

- **研究背景**：无监督强化学习旨在无外在奖励信号的情况下预训练可扩展的技能条件策略，为下游控制任务提供基础。现有工作已取得进展，但主要基于离策略（off-policy）方法以提高数据效率。
- **核心问题**：作者指出现有离策略URL方法存在两个被忽视的关键瓶颈：
  - **非平稳技能语义**：在离策略学习中，技能标签 \(z\) 存储在经验回放缓冲区，但其语义随表示函数 \(\phi\) 的演化而改变。同一技能标签 \(z\) 在不同训练阶段可能对应不同的行为轨迹，导致“语义漂移”，使策略更新受陈旧样本干扰，降低数据效率。
  - **脆弱的泛化能力**：技能策略通常直接以完整状态观测为输入，其中包含全局上下文信息（如绝对坐标、背景视觉特征）。这导致技能执行与全局上下文纠缠，当预训练与部署的全局分布发生偏移时，同一技能会产生不一致的行为，限制技能在下游任务中的复用。
- **整体含义**：解决这两个瓶颈是实现可扩展URL的关键，需要同时提升预训练的数据效率和策略对分布偏移的鲁棒性。

## 2. 方法论：核心思想、关键技术细节与算法流程

- **整体框架**：GenDa（Generalizable Data-efficient Agent）是一个统一的无监督RL框架，基于先前最先进的METRA（Park et al., 2024）构建，包含两个核心组件：技能重标记（Skill Relabeling）和互补信息瓶颈（CIB）。

### 2.1 技能重标记（Skill Relabeling）

- **动机**：缓解语义漂移。传统离策略URL使用固定的 \(z_{\text{roll}}\) 作为轨迹标签，当表示函数 \(\phi\) 更新后，旧标签变得过时。
- **实现细节**：
  - **z步（重标记）**：对于每条轨迹 \(\tau = (s_0, \dots, s_T)\)，根据当前表示函数 \(\phi_{\text{tgt}}\)（使用EMA网络）计算新的标签：
    \[
    z_{\text{relab}} = \text{unit}(\phi_{\text{tgt}}(s_T) - \phi_{\text{tgt}}(s_0))
    \]
    其中 \(\text{unit}(x) = x / \max(\|x\|_2, \epsilon)\)。
  - **\(\phi\)步（优化）**：使用重标记后的 \(z_{\text{relab}}\) 更新表示 \(\phi\)（停止梯度），优化目标：
    \[
    \sup_{\pi, \phi} \mathbb{E}_{p(\tau, z_{\text{relab}})} \left[ \sum_{t=0}^{T-1} (\phi(s') - \phi(s))^\top z_{\text{relab}} \right]
    \]
    约束：\(\|\phi(s) - \phi(s')\|_2 \leq 1\) 对所有相邻状态对成立。
  - **均匀性正则化**：防止表示坍塌，额外最小化对比损失：
    \[
    L_{\text{unif}} = \frac{1}{B} \sum_{i=1}^{B} \log \sum_{j \neq i} \exp(v_i^\top v_j)
    \]
    其中 \(v_i = \text{unit}(\phi(s_T^{(i)}) - \phi(s_0^{(i)}))\)，鼓励重标记技能向量在超球面上均匀分布。
  - **总体表示学习损失**：\(L_\phi = L_{\text{base}} + \beta L_{\text{unif}}\)，\(\beta=1.0\)。
- **策略重标记**：在策略学习阶段，从同一轨迹中采样不同跨度 \(c\) 生成 \(z_c = \text{unit}(\phi(s_{t+c}) - \phi(s_t))\)，并与 \(z_{\text{relab}}\) 和原始 \(z_{\text{roll}}\) 混合，丰富离策略奖励信号。

### 2.2 互补信息瓶颈（CIB）

- **动机**：使策略关注自我中心特征，忽略全局上下文。
- **实现**：训练一个编码器 \(q_\psi(\ell | s)\) 生成嵌入 \(\ell\)，与表示 \(\phi(s)\) 一起输入解码器 \(p_\omega(\hat{s} | \ell, \phi(s))\)。目标函数：
    \[
    J_{\text{CIB}} = I(\ell; s) - I(\ell; \phi(s))
    \]
    直觉上，解码器可利用 \(\phi(s)\) 重建已捕获的信息，从而 \(\ell\) 被迫保留互补信息（如自我中心感觉、关节状态等），丢弃全局上下文。
- **策略输入替换**：将原始状态 \(s\) 替换为 \(\ell \sim q_\psi(\ell | s)\)，即 \(\pi(a | \ell, z)\)。CIB在数据收集、训练和部署中一致使用。
- **算法流程**（Algorithm 1）：初始化 \(\pi, \phi, \phi_{\text{tgt}}, q_\psi, \lambda\) 和回放缓冲区；每个epoch内收集数据；对每个梯度步：1) 从缓冲区采样批次；2) 重标记技能标签（Eq.4）和跨度标签（Eq.8）；3) 更新 \(\phi\) 和拉格朗日乘子；4) 混合批次更新策略 \(\pi\)（使用SAC）；5) 更新CIB编码器；6) 更新EMA目标网络。

## 6. 主要结论与发现

1. **技能重标记显著提升数据效率**：相比METRA等基线，GenDa在几乎所有环境中更快达到更高覆盖率，尤其在Dog、Fish等难以学习的高维环境中首次成功。
2. **CIB提升了泛化能力**：在(RS, RG)和Maze等涉及全局偏移的任务中，GenDa成功率远超基线（例如在Humanoid-MazeHard上达到41%，而基线均为0%）。
3. **消融实验确认组件互补性**：去除Relabel后预训练覆盖率大幅下降；去除CIB后下游泛化显著退化，但覆盖率不受影响，说明两个组件分别解决不同瓶颈。
4. **高UTD利用率**：GenDa在UTD=0.5时性能稳定，而METRA在高UTD时反而下降，验证了技能重标记消除了语义漂移导致的梯度噪声。
5. **像素环境下同样有效**：在Humanoid-Pixels和Quadruped-Pixels上，GenDa在覆盖率和下游任务上均优于基线，CIB对视觉背景扰动也提供了一定鲁棒性。

## 7. 优点

- **方法创新性强**：
  - 首次形式化并解决了离策略URL中的语义漂移问题，提出技能重标记并附加均匀性正则化避免坍塌。
  - 创新性地将信息瓶颈思想用于解耦技能策略的输入，使策略依赖自我中心特征，提升了泛化鲁棒性。
- **实验全面且说服力强**：包括多个域、多种任务变体、全面的消融、超参数鲁棒性测试、UTD分析等，验证了方法的有效性与稳定性。
- **数据效率突出**：仅需2M步即可达到基线10M步的覆盖性能，适合真实机器人等交互昂贵场景。
- **兼容性好**：CIB作为即插即用模块，可集成到其他度量感知URL方法中；算法基于现有高效框架（JAX+EMA目标）。

## 8. 不足与局限

1. **假设限制**：技能重标记基于“每episode单一技能”假设，当episode内包含异质行为（如切换技能）时可能不准确。文中承认这一局限，并指出未来可扩展为时间变化标签。
2. **依赖度量感知表示**：CIB的有效性依赖于所学的度量感知 \(\phi\)，若应用于非度量感知URL方法（如DIAYN），需额外适配。这限制了通用性。
3. **操作任务不适用**：度量感知目标最大化时间距离，天然鼓励大范围运动，不利于精细操作。未测试低维或高精度控制任务（如机器人抓取）。
4. **视觉背景鲁棒性有限**：在像素环境下进行大幅背景变化测试（Unicolor/Gradation）时，CIB虽有提升，但覆盖率仍下降（从331降至206），说明完全解耦视觉背景仍有挑战。作者未使用图像增强，可能低估了结合数据增强的潜力。
5. **计算资源未报告**：缺乏GPU型号、训练时间等细节，影响可复现性评估。
6. **缺少跨任务转移实验**：下游任务与预训练环境相同（只是目标位置变化），未测试跨不同环境/机器人的迁移能力，而这是URL的核心目标之一。
7. **伦理声明**：论文认为无负面影响，但需注意预训练策略可能被误用于不良任务。
