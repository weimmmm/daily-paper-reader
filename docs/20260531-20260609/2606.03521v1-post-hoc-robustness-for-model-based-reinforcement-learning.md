## 摘要
为了提高强化学习（RL）在现实世界中的适用性，对抗鲁棒强化学习领域研究如何在对抗环境扰动下训练智能体。在该设定中，主角智能体在对手的环境扰动下优化策略，形成零和马尔可夫博弈。当对抗鲁棒强化学习与基于模型的RL结合时，对手可以针对学习到的转移模型而非训练环境。延伸这一思路，本工作引入了深度RL智能体在推理时的后验鲁棒化。通过将学习到的模型与训练好的名义策略结合，我们的方法执行鲁棒策略改进步骤，目标是在无需额外训练神经网络的情况下提升鲁棒性。具体而言，我们利用对抗轨迹展开下的模型预测控制，这些轨迹通过有界不确定性集内的投影梯度下降近似得到。此外，这些离线轨迹展开在考虑并缓解分布外问题的同时进行。通过在受扰动的Gymnasium MuJoCo环境中评估算法，同时考虑后验推理设定的计算限制，所提方法通过展示鲁棒性的显著改进得到了验证。

## Abstract
To improve the real-world applicability of reinforcement learning (RL), the field of adversarially robust RL studies how to train agents under adversarial environment perturbations. In this setting, a protagonist agent optimizes a policy under environmental perturbations from an adversary, resulting in a zero-sum Markov game. When adversarially robust RL is combined with model-based RL, the adversary can target a learned transition model instead of the training environment. Extending this idea, this work introduces post-hoc robustification of deep RL agents at inference time. By using the learned model in combination with a trained nominal policy, our approach performs a robust policy improvement step. The goal is to improve robustness without any additional training of neural networks. Specifically, we utilize model-predictive control under adversarial rollouts, which are approximated via projected gradient descent within a bounded uncertainty set. Furthermore, these offline rollouts are performed while considering and mitigating out-of-distribution issues. The proposed methodology is validated by demonstrating significant improvements in robustness when the algorithm is evaluated in perturbed Gymnasium MuJoCo environments, while considering the computational limitations of the post-hoc inference setting.

- **整体含义**：本文提出一种后验鲁棒化框架，允许用户在需要时（如部署到实际环境）动态定义不确定性集和鲁棒性需求，而不牺牲训练时的性能。

### 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

- **核心思想**：利用已学习的转移模型和名义价值函数，在推理时通过**对抗性模型预测控制**进行鲁棒策略优化。在每个时间步，执行带悲观扰动的模型展开，然后使用MPPI优化当前策略。
- **关键技术细节**：
    - **对抗性展开**：对于模型预测的下一状态 $s'$，使用投影梯度下降（PGD）添加扰动 $\delta$，最小化名义价值函数 $V^\pi_\psi(s'+\delta)$，并约束 $\|\delta\|_2 \le r_2$（L2球内），对应最优传输成本（OTC）不确定性集。
    - **悲观价值估计**：递归定义 $V^{\text{rollout}}_k(s_k) = r(s_k,\pi(s_k)) + \gamma V^{\text{rollout}}_{k+1}(s_{k+1})$，其中 $s_{k+1} \sim \arg\min_{P\in\mathcal{P}} \mathbb{E}_{s'\sim P(\cdot|s,a)}[V^\pi(s')]$，即每次转移都选择最坏情况下的下一状态价值。
    - **OOD缓解**：使用集成模型（ensemble）的方差作为认知不确定性指标，当方差超过阈值 $h$ 时截断展开，避免模型在分布外区域被过度利用。
    - **MPC优化**：使用MPPI算法近似求解 $\pi^{\text{MPC}}_s = \arg\max_{\pi_s} \mathbb{E}_{\pi_s}[V^{\text{rollout}}(s)]$，并以原策略作为初始化。
- **公式说明**：
    - 扰动更新：$\delta \leftarrow \text{Proj}_{\|\delta\|_2\le r_2} \left( \delta - \alpha \frac{\nabla_{(s'+\delta)} V_\psi(s'+\delta)}{\|\nabla_{(s'+\delta)} V_\psi(s'+\delta)\|_2} \right)$
    - 投影操作：$\text{Proj}_{\|\delta\|_2\le r_2}(\delta) = \delta \cdot \min(1, r_2/\|\delta\|_2)$
    - 滚动深度 $N$ 和不确定集半径 $r_2$ 是手动调参的超参数。

### 6. 论文的主要结论与发现

1. 本文提出的后验鲁棒化方法**显著提升了MBRL智能体在扰动环境下的鲁棒性**，在所有测试环境下性能下降幅度远小于基线。
2. **对抗性扰动是关键**：Non-Robust MPC（无扰动）在大部分情况下性能接近基线甚至更差，而Robust MPC大幅改进，证实了悲观展开的必要性。
3. **OOD缓解不可或缺**：在Reacher-v4上，若不使用epistemic截断（Fig 2），性能会崩坏至基线以下。
4. **滚动深度 $N$ 和不确定集半径 $r$** 影响鲁棒性：更长滚动和适中半径能提升鲁棒性，但过大会导致模型误差累积（Hopper-v4中 $N>70$ 时崩溃）。
5. **推理时间可接受**：在高端GPU上每步约0.1s（Hopper），0.02s（Reacher），适合离线或低实时性场景。

### 7. 优点：方法或实验设计上的亮点

- **无需额外训练**：完全在推理阶段进行鲁棒改进，避免了对抗训练的代价和不稳定性。

- **计算成本评估**：提供了不同GPU下的推理时间，对实际部署有参考价值。
