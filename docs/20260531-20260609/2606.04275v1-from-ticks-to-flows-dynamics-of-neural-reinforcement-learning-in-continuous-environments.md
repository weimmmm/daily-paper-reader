## 摘要
我们提出了一种新颖的深度学习强化学习理论框架，通过将问题建模为连续时间随机过程，借鉴随机控制的理论。在先前工作的基础上，我们引入了一种可行的演员-评论家算法模型，该模型同时考虑了探索和随机转移。对于单隐藏层神经网络，我们展示了环境状态可以表述为两个时间尺度的过程：环境时间和梯度时间。在此框架内，我们描述了表示环境状态的时变随机变量以及累积折扣回报的估计如何在梯度步骤中演化，在两层网络的无限宽度极限下。利用随机微分方程理论，我们首次在连续强化学习中推导出一个方程，描述了在极小的学习率下，每个梯度步骤中状态分布的无穷小变化。总体而言，我们的工作为研究过参数化神经演员-评论家算法提供了新的非参数化表述。我们通过一个简单的连续控制任务实验验证了我们的理论结果。

## Abstract
We present a novel theoretical framework for deep reinforcement learning (RL) in continuous environments by modeling the problem as a continuous-time stochastic process, drawing on insights from stochastic control. Building on previous work, we introduce a viable model of actor-critic algorithm that incorporates both exploration and stochastic transitions. For single-hidden-layer neural networks, we show that the state of the environment can be formulated as a two time scale process: the environment time and the gradient time. Within this formulation, we characterize how the time-dependent random variables that represent the environment's state and estimate of the cumulative discounted return evolve over gradient steps in the infinite width limit of two-layer networks. Using the theory of stochastic differential equations, we derive, for the first time in continuous RL, an equation describing the infinitesimal change in the state distribution at each gradient step, under a vanishingly small learning rate. Overall, our work provides a novel nonparametric formulation for studying overparametrized neural actor-critic algorithms. We empirically corroborate our theoretical result using a toy continuous control task.

- **研究动机**：深度强化学习在连续状态-动作空间的控制任务中取得了巨大成功，但其理论基础严重滞后于经验进展。与深度监督学习不同，RL 中智能体的策略更新会导致环境数据分布（状态分布）随之改变，这给理论分析带来根本困难。现有深度 RL 理论多局限于离散时间、表格或特定假设，缺乏对连续环境中过参数化神经网络 Actor-Critic 学习动力学的通用刻画。
- **核心问题**：如何描述在连续时间环境、梯度步尺度上，由神经网络参数化 Actor 和 Critic 的 Agent 的状态、动作和值估计的联合演化？
- **整体含义**：本文首次建立了深度 RL 在连续控制中的“双时钟”随机动力学框架——环境时间（瞬间变化）和梯度时间（参数更新）。通过随机微分方程和伊藤-泰勒展开，推导出状态分布随梯度步演化的封闭非参数方程，为理解过参数化神经 Actor-Critic 算法提供了严格的理论基础。

## 二、方法论：核心思想、关键技术细节、公式与算法流程

### 核心思想
将连续时间 RL 中的环境转移建模为随机微分方程（SDE），同时引入来自策略探索的额外噪声。将 Actor 和 Critic 用单隐层线性化神经网络（NNGP/NTK 意义上的无限宽度极限）参数化，通过伊藤-泰勒展开将状态变量表示为网络权重的多项式。利用宽度无穷大时网络输出趋近高斯过程的性质，结合鞅中心极限定理和条件大数定律，推导出梯度步下状态、动作、值估计及其时间/空间导数的联合演化方程。

### 关键技术细节
1. **连续时间 MDP 的定义**：控制仿射形式的 SDE：  
   \[ ds_t = (g(s_t) + h(s_t)a_t)dt + \sigma(s_t)dw_t \]  
   其中 \(w_t\) 是 Wiener 过程，\(g,h,\sigma\) 光滑。策略为确定性反馈控制 \(\pi(s)\)。

2. **探索动力学**：为解决标准探索中噪声维度增加的问题，提出等价数值格式：  
   \[ d\tilde{s}_t^\pi = (g(\tilde{s}_t^\pi) + h(\tilde{s}_t^\pi)\pi(\tilde{s}_t^\pi))dt + \sqrt{h(\tilde{s}_t^\pi)^2 + \sigma(\tilde{s}_t^\pi)^2}dw_t \]  
   将探索噪声和环境噪声合并为单一 Wiener 过程（Lemma 3.1 证明弱收敛等价性）。

3. **线性化神经网络**：Actor 和 Critic 均采用单隐层线性化近似（只训练第一层权重 \(W\)，固定第二层 \(C\)）：  
   \[ F_{\text{lin}}^\pi(s;W) = F_\pi(s;W^0) + \Phi(s;W^0)(W - W^0) \]  
   其中 \(\Phi\) 是 NTK 特征。学习率 \(\eta = O(1/\sqrt{n})\)。

4. **梯度更新**（算法1）：在线收集一条轨迹 \(s_{t_i}, r_{t_i}\)，计算时序差分误差 \(\delta_i = \partial_t v + r - \beta v\)，然后用 Riemann 和近似连续时间梯度更新 Actor 和 Critic 参数。

5. **伊藤-泰勒展开**：将状态变量 \(s_{t,\tau}\) 展开为关于参数变化 \(\Delta W^\tau\) 的无穷级数多项式（Sec. C, F）。通过“留一法”分离对特定参数索引的依赖，证明除 \(O(\eta n^{-1/2})\) 的修正项外，状态可分解为独立部分和依赖部分。

6. **主定理（Theorem 6.1）**：在给定环境维度 \(d_s=d_a=1\)、平滑动力学、tanh 激活、i.i.d.初始化及学习率 \(\eta=O(1/\sqrt{n})\) 下，状态 \(s_{t,\tau}\)、动作 \(a_{t,\tau}\)、动作导数 \(a'_{t,\tau}\)、值估计 \(v_{t,\tau}\) 和时间导数 \(v'_{t,\tau}\) 构成封闭系统。这些变量在单个梯度步的变化可表达为高斯分布（均值和方差都仅依赖于该组变量当前值的函数），加上一个 \(O(1/\sqrt{n})\) 的误差项。特别地，状态变化 \(\Delta s_{t,\tau}\) 由积分表达式给出，包含了通过伊藤-泰勒展开导出的核函数 \(Z_{t,l,\tau}\)。
